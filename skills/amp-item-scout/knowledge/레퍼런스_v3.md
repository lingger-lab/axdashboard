## 4. 심층 검증 체크리스트 (Phase 2: DEEP)

상위 5건 각각에 대해:

| 검증 항목 | 방법 | 필수 여부 |
|---|---|---|
| 해외 팔리는 증거 | web_search + **web_fetch** | ★ 필수 |
| 한국 팔리는 증거 | web_search + **web_fetch (크몽 등)** | ★ 필수 |
| 불만 증거 | web_search | ○ 권장 |
| 킬 리스크 | web_search | ★ 필수 |
| 실제 가격 확인 | web_fetch (서비스 페이지) | ○ 권장 |

**DEEP 품질 기준:**
- web_fetch 최소 2회 사용 (검색 스니펫만으로는 깊이 부족)
- 숫자(MRR, 리뷰 수, 이용자 수)가 있으면 반드시 포함
- 없는 데이터를 만들지 않는다 → "근거 없음"
- [실시간검색] / [추정] 태그 필수

## 6. v2 → v3 변경 이력

| 항목 | v2 | v3 |
|---|---|---|
| 소스 수 | 23개 | **30개** (+7) |
