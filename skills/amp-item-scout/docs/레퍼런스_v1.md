### 필요역량키워드(J열) 작성 규칙

이 열이 의뢰자 매칭의 핵심이다.

- 고정 카테고리를 쓰지 않는다
- 의뢰자가 자기 경력을 설명할 때 쓸 법한 **실제 단어**를 사용
- 한 아이템에 3~7개 키워드를 쉼표로 나열
- 의뢰자가 "물류관리 20년"이라고 하면, J열에 "물류"가 포함된 아이템이 매칭됨

키워드 예시:
- 세무신고, 재무제표, 원가분석, 부가세, 자금관리
- 품질검사, 공정관리, ISO, 설비보전, 납기관리
- 고객응대, CS관리, 컴플레인처리, 콜센터
- 콘텐츠기획, SNS운영, 카피라이팅, 브랜딩
- 물류, 재고관리, 입출고, 발주, 배송
- 영업, 거래처관리, 견적, 수주, 납품
- 강의, 커리큘럼설계, 학생관리, 교재개발
- 특허, 기술문서, R&D기획, 시제품, 인증
- 인사관리, 급여, 채용, 노동법, 4대보험
- 농산물유통, 시세분석, 직거래, 가공, 브랜딩
- 경영기획, 사업계획, 전략수립, 시장분석, 투자유치
- 웹디자인, UI설계, 사용자경험, 프로토타입
- 요식업, 메뉴개발, 원가관리, 주방관리, 위생
- 부동산, 임대관리, 시세분석, 계약, 중개
- 의료, 환자관리, 진료보조, 의료기기, 건강관리

## 3. 예시 아이템 카드 5건

### IT001: 창업기업 세무 안심 점검
- 고객: 창업 1~3년차 소기업
- 고통: "부가세 신고 제대로 한 건지 불안"
- AI 80%: 거래 자동 분류, 신고 기한 알림, 이상치 탐지
- 사람 20%: 월 1회 원격 점검, 이상치 판단, 긴급 자문
- 필요역량키워드: 세무신고, 재무제표, 부가세, 원가분석, 자금관리
- 필요경력: 재무회계 10년+
- 해외증거: Bench.co MRR $2M+ [실시간검색]
- 한국증거: 삼쩜삼 이용자 1,500만+ [실시간검색]
- 불만증거: 네이버카페 "세무사 비용 부담" 월 50건+
- 킬리스크: 세무사 저가 구독 서비스 등장 시

### IT002: AI 견적서·제안서 자동 생성
- 고객: 제조업·서비스업 영업 담당자
- 고통: "견적서 매번 처음부터 만든다, 실수도 잦다"
- AI 80%: 과거 견적 패턴 학습, 품목·단가·조건 자동 채움
- 사람 20%: 가격 전략 검토, 경쟁 입찰 시 차별화 포인트
- 필요역량키워드: 영업, 견적, 수주, 거래처관리, 납품, 가격협상
- 필요경력: 영업 10년+
- 해외증거: PandaDoc ARR $100M+ [실시간검색]
- 한국증거: 크몽 "견적서 작성" 서비스 다수 거래 [실시간검색]
- 불만증거: 블라인드 "견적서 만드느라 야근" 게시물 다수
- 킬리스크: CRM 솔루션에 견적 기능 기본 탑재 추세

### IT003: 멀티채널 콘텐츠 자동 생산
- 고객: SNS 마케팅 인력이 없는 자영업·소기업
- 고통: "SNS 해야 하는 건 아는데 뭘 올려야 할지 모르겠다"
- AI 80%: 카드뉴스·블로그·SNS 텍스트 자동 생성, 멀티채널 변환
- 사람 20%: 업종별 톤 설정, 핵심 메시지 검수, 전략 방향
- 필요역량키워드: 콘텐츠기획, SNS운영, 마케팅, 카피라이팅, 브랜딩, 홍보
- 필요경력: 마케팅/홍보 5년+
- 해외증거: Buffer, Hootsuite 수천만 사용자 [실시간검색]
- 한국증거: 크몽 "SNS 대행" 월 500건+ 거래 [추정]
- 불만증거: 네이버카페 "마케팅 대행 비용 부담" 게시물 다수
- 킬리스크: 고객이 직접 ChatGPT로 만드는 시대 도래

### IT004: 온라인 코칭 플랫폼 구축 대행
- 고객: 전문성은 있지만 디지털 마케팅 역량이 없는 시니어
- 고통: "내 전문성으로 온라인 강의를 하고 싶은데 방법을 모른다"
- AI 80%: 랜딩페이지 자동 생성, 결제 연동, 수강생 관리
- 사람 20%: 가격 전략, 마케팅 퍼널 설계, 콘텐츠 구조
- 필요역량키워드: 교육기획, 커리큘럼, 온라인강의, 코칭, 컨설팅, 퍼스널브랜딩
- 필요경력: 교육/강의 경험 5년+ 또는 특정 분야 전문성
- 해외증거: Kajabi ARR $100M+, Teachable 수백만 크리에이터 [실시간검색]
- 한국증거: 클래스101 누적 수강 1,000만+ [실시간검색]
- 불만증거: 네이버카페 "온라인 강의 플랫폼 수수료 너무 높다"
- 킬리스크: 노코드 도구 발달로 직접 구축 가능해지는 추세

### IT005: 소기업 채용 스크리닝 AI
- 고객: 직원 5~20명 규모 소기업
- 고통: "이력서 50개 왔는데 볼 시간이 없다"
- AI 80%: 이력서 자동 파싱, 직무적합도 점수화, 면접 질문 생성
- 사람 20%: 최종 3인 숏리스트 검토, 조직 적합성 판단
- 필요역량키워드: 채용, 인사관리, 면접, HR, 조직관리, 인력운영
- 필요경력: 인사/HR 7년+
- 해외증거: Manatal MRR $500K+ [추정]
- 한국증거: 원티드, 사람인 AI 매칭 도입 [실시간검색]
- 불만증거: 블라인드 "채용 스크리닝에 시간 너무 많이 든다"
- 킬리스크: 잡플랫폼(사람인, 잡코리아)이 AI 스크리닝 기본 제공 시

## 4. 의뢰자 매칭 예시

**입력:** "물류관리 20년, 재고 최적화가 특기"

**매칭 프로세스:**
1. 키워드 추출: 물류, 재고, 관리, 최적화, 입출고
2. J열(필요역량키워드) 검색
3. 매칭 결과: 해당 키워드가 포함된 아이템 제시

**출력 예시:**
```
━━━━━━━━━━━━━━━━━━━━━━━━━━
  물류관리 20년 경력에 맞는 아이템
━━━━━━━━━━━━━━━━━━━━━━━━━━

[1] 소기업 재고·발주 자동 관리
    고객: 직원 5~20명 유통/제조업
    고통: "재고 파악이 안 돼서 과잉발주가 반복된다"
    AI 80%: 재고 자동 집계, 발주점 알림, 수요 예측
    당신의 한 끗: 재고 수치만 보면 어디서 누수 나는지 판단
    ─────
    해외: Cin7 ARR $50M+ [실시간검색]
    한국: 크몽 "재고관리 자동화" 거래 다수 [실시간검색]

[2] ...
[3] ...
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 5. 주간 수집 체크리스트

매주 월요일 실행 시 아래 순서를 따른다:

```
□ Step 1: "이번 주 아이템 스카우트 해줘" 입력
□ Step 2: 결과 확인 — 아이템 5~10건 + 시장 근거 3~5건
□ Step 3: Google Sheets 저장 확인
□ Step 4: 필요 시 추가 소스 지정 ("유튜브도 검색해줘")
□ Step 5: 킬 테스트 필요한 아이템 추가 검증
```

소요 시간: 약 20~30분
