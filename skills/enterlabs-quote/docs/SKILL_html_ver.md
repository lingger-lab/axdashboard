# ENTERLABS 견적서 자동생성 스킬

## 역할
ENTERLABS의 AI 솔루션 견적서를 자동 생성합니다. 
사용자의 자연어 입력을 구조화된 견적 데이터로 변환하고,
HTML 견적서 파일 또는 Markdown 요약으로 출력합니다.

---

## Step 1 — 입력 정보 수집

사용자 메시지에서 아래 정보를 추출합니다.
누락된 필수 정보는 짧게 질문해서 보완하세요.

### 필수 정보
| 항목 | 설명 | 예시 |
|------|------|------|
| `client_name` | 고객사명 | 창원시청 |
| `items[]` | 견적 항목 목록 | (아래 참조) |

### 선택 정보 (없으면 기본값 사용)
| 항목 | 기본값 | 설명 |
|------|--------|------|
| `project_name` | 미정 | 프로젝트/솔루션명 |
| `quote_date` | 오늘 날짜 | 견적일 |
| `valid_days` | 30일 | 견적 유효기간 |
| `vat_included` | true | 부가세 10% 포함 여부 |
| `currency` | KRW | 통화 |
| `contact_name` | 빈칸 | 담당자 성함 |
| `notes` | 기본 결제조건 | 특이사항 |

### 항목(item) 구조
```
category    : 'AI 솔루션' | '개발' | '컨설팅' | '유지보수' | '교육' | '기타'
description : 항목명 및 상세 내용
qty         : 수량 (숫자)
unit        : '식' | '개월' | '시간' | '일' | '건' | '라이선스' | '세트'
unit_price  : 단가 (숫자, 원 기준)
```

## Step 2 — 가격 산정 가이드라인

사용자가 금액을 명시하지 않은 경우, 아래 기준을 참고하여 합리적 가격을 제안하세요.
(모든 가격은 VAT 별도, 단위: 원)

```

· LLM API 연동 구축           : 2,000,000 ~ 5,000,000 / 식
· 데이터 파이프라인 구축       : 3,000,000 ~ 10,000,000 / 식
· AI 에이전트 개발             : 5,000,000 ~ 15,000,000 / 식

[개발]
· 웹/앱 프론트엔드 개발        : 2,000,000 ~ 8,000,000 / 식
· 백엔드 API 개발              : 2,000,000 ~ 6,000,000 / 식
· 데이터베이스 설계            : 1,000,000 ~ 3,000,000 / 식
· 시스템 통합 (Integration)    : 1,500,000 ~ 5,000,000 / 식

[컨설팅]
· AI 도입 전략 컨설팅          : 500,000 ~ 1,500,000 / 시간
· 디지털 전환 컨설팅           : 3,000,000 ~ 8,000,000 / 식
· 데이터 분석 리포트           : 1,500,000 ~ 4,000,000 / 건

[유지보수]
· 기본 유지보수                 : 200,000 ~ 500,000 / 개월
· 고급 유지보수 (SLA 포함)     : 500,000 ~ 1,500,000 / 개월
· 모니터링 & 운영              : 300,000 ~ 800,000 / 개월

· 팀 대상 AI 도구 교육        : 500,000 ~ 1,000,000 / 회
```

> 💡 복합 프로젝트의 경우 20~30% 패키지 할인 적용 가능하다고 언급하세요.

## Step 3 — 출력 결정

사용자의 요청 맥락에 따라 출력 방식을 선택합니다.

### 출력 A: HTML 견적서 파일 (기본)
- "파일로", "다운받게", "HTML", "인쇄" 키워드가 있을 때
- 또는 항목이 3개 이상일 때 (복잡한 견적)
- → `assets/enterlabs_quote.html` 템플릿을 기반으로 생성 (하단 참조)

### 출력 B: Markdown 견적 요약 (빠른 확인용)
- "간단히", "빠르게", "대략", "이 정도면?" 같은 표현이 있을 때
- 또는 항목이 1~2개인 단순 견적
- → 마크다운 표 형태로 출력

---

## Step 4A — HTML 파일 생성 방법

`assets/enterlabs_quote.html` 파일을 기반으로 합니다.

### 수행 순서
1. `view /mnt/skills/user/enterlabs-quote/assets/enterlabs_quote.html` 로 템플릿 읽기
2. JavaScript `init()` 함수 내 데이터를 사용자 데이터로 교체
3. `/mnt/user-data/outputs/ENTERLABS_[견적번호].html` 로 저장
4. `present_files` 로 파일 전달

### 핵심 교체 위치 (JavaScript init 함수)
```javascript
// 견적 번호
document.getElementById('quoteNo').value = '{{quote_number}}';

// 날짜
document.getElementById('quoteDate').value = '{{quote_date}}';
document.getElementById('validDate').value = '{{valid_date}}';

// 고객사
// party-name 두 번째 input (수신자)
inputs[1].value = '{{client_name}}';

// 담당자
prInputs[0].value = '{{contact_name}}';

// 프로젝트명
document.querySelector('.proj-title').value = '{{project_name}}';

// 프로젝트 설명
document.querySelector('.proj-desc').value = '{{project_desc}}';

// 항목 추가 (addRow 반복 호출)
addRow({cat:'{{category}}', desc:'{{description}}', qty:{{qty}}, unit:'{{unit}}', price:{{unit_price}}});
```

## Step 4B — Markdown 견적 요약 포맷

```markdown
## 견적서 — ENTERLABS × {{client_name}}

**프로젝트**: {{project_name}}  
**견적일**: {{quote_date}} | **유효기간**: {{valid_date}}까지

| # | 구분 | 항목 | 수량 | 단위 | 단가 | 금액 |
|---|------|------|------|------|------|------|
| 1 | AI 솔루션 | ... | 1 | 식 | 5,000,000 | 5,000,000 |

**공급가액**: ₩ X,XXX,XXX  
**부가세 (10%)**: ₩ XXX,XXX  
**합계금액**: **₩ X,XXX,XXX**

> 결제조건: 계약금 50% / 잔금 50% (납품 시)  
> 본 견적은 발행일로부터 30일간 유효합니다.
```

## 품질 체크리스트

출력 전 아래를 확인하세요:
- [ ] 견적번호 형식: `EL-YYYYMMDD-XXXX`
- [ ] 금액이 0이 아닌 실질적 값
- [ ] VAT 포함/별도 명확히 표기
- [ ] 고객사명 정확 기재
- [ ] 항목별 단위(식/개월/시간 등) 적절히 선택
- [ ] 합계 계산 정확성 (소계 + VAT = 합계)
