## ━━ 기본 Q (G1~G13): 양자구도 핵심 분석 ━━

### [G1] 구도 진단 — "이 선거는 어떤 싸움인가?"

**분석 질문**: 이 선거의 핵심 대결 구도를 진단하라.

생성 항목:
- **구도 유형 판정**: 이념 대결형 / 인물 대결형 / 정책 대결형 / 복합형 중 어디에 해당하는가
- **지배 프레임 진단**: 현재 어떤 프레임이 유권자 인식을 지배하는가
- **프레임 유불리**: 국정 심판론 vs 지역 발전론 — 각 후보에게 유리한 프레임
- **결정적 변수 TOP3**: 최종 승패를 가를 3가지 변수
- **구도 변화 시나리오**: 선거 전까지 구도가 바뀔 수 있는 조건

### [G2] 여당 후보(A) 포지셔닝

**분석 질문**: 여당 후보의 본선 포지셔닝 전략을 설계하라.

생성 항목:
- **핵심 포지셔닝**: "이 사람은 OO" — 유권자 인식 속 포지션 목표
- **강점 활용 시나리오**: 집권당 자원을 최대화하는 3가지 상황
- **약점 방어 전략**: 국정 지지율 연동 리스크 최소화 방법
- **차별화 포인트**: 야당 후보(B)와 비교 시 유일한 것 한 문장
- **메시지 계층**: 3초 / 10초 / 30초 유권자 대상 메시지

### [G3] 야당 후보(B) 포지셔닝

**분석 질문**: 야당 후보의 본선 포지셔닝 전략을 설계하라.

생성 항목:
- **핵심 포지셔닝**: "이 사람은 OO" — 유권자 인식 속 포지션 목표
- **강점 활용 시나리오**: 심판론·인물론을 최대화하는 3가지 상황
- **약점 방어 전략**: 정책 실행력 검증 요구 대응
- **차별화 포인트**: 여당 후보(A)와 비교 시 유일한 것 한 문장
- **메시지 계층**: 3초 / 10초 / 30초 유권자 대상 메시지

### [G4] 부동층 분석 — "누가 이 선거를 결정하는가?"

**분석 질문**: 부동층의 규모, 성향, 이동 방향을 분석하라.

생성 항목:
- **부동층 규모 추정**: 현재 미결정 유권자 비율과 근거
- **부동층 프로파일**: 연령/지역/직업/이슈 관심사 특성
- **이동 조건 분석**: 어떤 사건/메시지가 부동층을 움직이는가
- **각 후보별 부동층 공략 전략**: A 후보 접근법 vs B 후보 접근법
- **부동층 흡수 최적 시점**: D-60 / D-30 / D-14 단계별 공략 포인트

### [G5] 정당 지지율 vs 후보 개인 지지율

**분석 질문**: 정당 지지율과 후보 개인 지지율의 괴리를 분석하고 전략적 시사점을 도출하라.

생성 항목:
- **괴리 측정**: 정당 지지율 vs 후보 개인 지지율 갭 (데이터 또는 추론 기반)
- **A 후보 분석**: 정당 프리미엄 활용 vs 개인 브랜딩 강화 — 어느 쪽이 유리한가
- **B 후보 분석**: 정당 프리미엄 활용 vs 개인 브랜딩 강화 — 어느 쪽이 유리한가
- **전략적 시사점**: 정당 색채 강화 vs 탈정당 인물론 전략 선택 기준
- **선거운동 방향**: 정당 행사 집중 vs 개인 유세 집중 최적 배분

### [G6] 지역 의제 분석 — "유권자가 진짜 원하는 것"

**분석 질문**: 해당 선거구 유권자가 가장 중요시하는 의제를 분석하라.

생성 항목:
- **의제 TOP5**: 유권자 관심 이슈 우선순위 (데이터/추론 기반)
- **의제별 양 후보 공약 정합성**: 유권자 관심 의제에 두 후보 공약이 부합하는가
- **의제 경쟁 우위**: 의제별 어느 후보의 공약이 더 설득력 있는가
- **생활정치 의제**: 일상에 직접 닿는 의제 — 교통/의료/교육/복지 등
- **의제 활용 전략**: 각 후보가 어떤 의제를 얼마나 부각해야 하는가

### [G7] 선거 환경 변수 모니터링

**분석 질문**: 선거 결과에 영향을 미칠 외부 변수를 식별하고 모니터링 체계를 설계하라.

생성 항목:
- **변수 8종 진단**:
  1. 국정 지지율 추이 (전국/지역 보정)
  2. 주요 경제 지표 (물가/실업/성장률)
  3. 지역 현안 이슈 변화
  4. 중앙당 지원 강도 차이
  5. 투표율 예측 변수
  6. 미디어 여론 흐름
  7. 돌발 스캔들/사건 리스크
  8. 제3후보 또는 단일화 변수
- **조기 경보 신호**: 각 변수의 이상 징후 감지 기준
- **트리거 포인트**: "이 신호 시 전략 변경" 의사결정 기준
- **시나리오 3종**: 여당 우세 / 초접전 / 야당 우세 조건

### [G8] 공약 실행력 검증 — "이게 진짜 되나?"

**분석 질문**: 양 후보의 핵심 공약 실행 가능성을 검증하라.

생성 항목:
- **가능/검토/불가 분류표**: 시장/단체장 권한 내 / 중앙 협의 필요 / 법률 개정 필요
- **공약별 실행 카드**: 배경·목표·예산추정·권한·절차·일정·KPI
- **Before→After**: 현재 수치 → 임기 말 목표 수치 (양 후보 비교)
- **실행력 경쟁 우위**: 의제별 어느 후보 공약이 더 실현 가능한가
- **유권자 체감 지수**: 공약이 일상에 닿는 정도 (1~5점 평가)

### [G10] 유권자 수요 매핑

**분석 질문**: INSIGHTS 데이터와 공개 통계를 교차 참조하여 유권자 수요 지도를 작성하라.

생성 항목:
- **세그먼트별 의제 우선순위**: 연령/지역/직업별 TOP3 관심 이슈
- **의제-공약 정합성 매트릭스**: 유권자 수요 × 양 후보 공약 연결 여부
- **정합성 약한 세그먼트**: 공약 또는 메시지 조정이 필요한 영역
- **INSIGHTS 이슈 TOP10**: 여론에서 가장 많이 언급된 이슈와 공약 매칭률
- **세그먼트별 "핵심 1문장 메시지"**: 각 세그먼트를 향한 최종 메시지 확정

### [G11] 메시지 계층 설계

**분석 질문**: 양 후보 각각의 본선 메시지를 3초/10초/30초 계층으로 설계하라.

생성 항목 (A 후보 / B 후보 각각):
- **3초**: 한 문장 + 사실 1개 (TV 자막/현수막용)
- **10초**: 문제→원인→해결책 (엘리베이터 피치)
- **30초**: 풀 스토리 (유세 도입부)
- **부동층 특화**: 중도층에게만 통하는 설득 메시지
- **지지층 결집**: 핵심 지지자를 투표장으로 끌어내는 감성 메시지

### [G12] 선거운동 채널 전략

**분석 질문**: 공식 선거운동 기간 채널별 전략을 설계하라.

생성 항목:
- **채널별 경쟁 우위 비교**: TV / 유튜브 / 인스타 / 카카오 / 지역 언론 / 직접 유세
- **채널별 콘텐츠 유형**: 각 채널에서 효과적인 포맷 (숏폼/토론클립/카드뉴스 등)
- **공식 선거운동 기간 캘린더**: D-13~D-1 일간 활동 계획
- **리퍼포징 체계**: 유세 1회 → 숏폼/카드/FAQ/블로그 변환
- **AI 활용 가이드**: 각 단계 효율화 도구

### [G13] TV 토론 전략

**분석 질문**: TV 토론 승리를 위한 전략과 답변 템플릿을 작성하라.

생성 항목:
- **예상 주제 TOP5**: 사회자 및 상대 후보가 제기할 이슈
- **주제별 답변 템플릿**: 팩트→원칙→조치→시간표 구조
- **양자 공방 시나리오**: 상대 공격 예상 + 방어 라인 + 역공 포인트
- **토론 후 콘텐츠**: 하이라이트→숏폼→카드→팩트체크 리퍼포징
- **위기 답변 준비**: 즉답이 필요한 예상 공격 5종

## ━━ 신규 Q (GN1~GN12): 양자구도 특화 분석 ━━

### [GN1] 투표율 시나리오 분석 🆕

**분석 질문**: 투표율 수준에 따른 양 후보 유불리를 분석하라.

생성 항목:
- **역대 투표율 패턴**: 해당 선거구 동종 선거 투표율 추이
- **투표율 구간별 유불리**:
  - 고투표율 (60%↑): 어느 후보에게 유리하고 그 이유는?
  - 중간 (50~60%): 각 후보의 전략
  - 저투표율 (45%↓): 어느 후보에게 유리하고 그 이유는?
- **투표율 관리 전략**: A 후보 vs B 후보 각각의 접근법
- **지지층 투표 독려 메시지**: 각 후보 지지자 대상 맞춤 메시지

### [GN2] 양자 대결 표 이동 시뮬레이션 🆕

**분석 질문**: 제3후보 사퇴, 단일화 등 구도 변화 시 표 이동을 시뮬레이션하라.

생성 항목:
- **현재 구도 기준선**: A 후보 vs B 후보 현재 지지율 갭
- **구도 변화 시나리오**:
  - 제3후보 등장 시: 표 분산 방향 및 비율
  - 단일화 압박 시: 협상 조건 및 표 귀속 시뮬레이션
  - 여론 급변 이벤트 시: 10%p 이상 이동 조건
- **표 이동 방어 전략**: 이탈 가능 지지층 방어 방법
- **부동층 흡수 시나리오**: 마지막 2주 내 부동층 이동 예측

### [GN3] 정당 자원 활용 전략 🆕

**분석 질문**: 양 후보가 소속 정당의 자원을 선거 승리에 활용하는 최적 방법을 설계하라.

생성 항목:
- **여당(국민의힘) 자원 목록**: 중앙당 지원 / 현직 의원 유세 / 정부 정책 연계 / 재정
- **야당(민주당) 자원 목록**: 중앙당 지원 / 현직 의원 유세 / 심판론 연대 / 재정
- **정당 색채 강도 전략**: 언제 정당을 부각하고 언제 개인을 부각하는가
- **중앙-지방 갈등 리스크**: 중앙당 개입이 오히려 역효과를 낼 조건
- **정당 이미지 레버리지**: 긍정 이미지 증폭 방법

### [GN4] 선거 로드맵 — D-60~D-day 🆕

**분석 질문**: 양 후보 각각의 선거 로드맵을 D-60~D-day까지 설계하라.

생성 항목:

| 단계 | 기간 | A 후보 핵심 활동 | B 후보 핵심 활동 | 전환 기준 |
|------|------|----------------|----------------|---------|
| 준비기 | D-60~D-45 | 조직 정비, 메시지 확정 | 조직 정비, 심판론 구체화 | 여론조사 5%p 이상 격차 |
| 본격기 | D-44~D-22 | 공약 발표, 유세 개시 | 공약 발표, 현장 밀착 | 부동층 이동 시작 감지 |
| 결전기 | D-21~D-7 | 부동층 흡수, 토론 승리 | 부동층 흡수, 토론 승리 | 격차 3%p 이내 |
| 마무리 | D-6~D-1 | 지지층 결집, 투표 독려 | 지지층 결집, 투표 독려 | D-1 기준 전략 고정 |

- **주차별 핵심 활동 1개씩** (D-60~D-day)
- **위기 발생 시 플랜 B**: 단계별 조정 방안

### [GN5] 네거티브 공격 대응 프로토콜 🆕

### [GN12] 비판적 자기검증 — "이 분석이 균형 잡혀 있는가?" 🆕

**분석 질문**: 전체 보고서에 대해 특정 후보 편향 여부를 검증하고 균형을 확보하라.

**⚠️ 실행 시점**: EXT-1~4 초안 완성 후, 최종 보고서 조립 전 반드시 실행

**5대 검증 영역:**

**① 구도 진단 편향**
- 현재 진단이 특정 정당에 유리한 전제를 깔고 있는가?
- "국정 심판론"을 기정사실로 전제하는가? (야당 유리 편향 리스크)
- "지역 발전 안정론"을 기정사실로 전제하는가? (여당 유리 편향 리스크)

**② 후보 분석 형평성**
- A 후보와 B 후보의 강점 분석 분량이 균형 잡혀 있는가?
- 한 후보의 약점이 더 상세하게 서술되지 않았는가?
- 두 후보에게 동일한 검증 기준이 적용되었는가?

**③ 세그먼트 편향**
- 특정 세그먼트(예: 청년, 노동자)에 과도하게 집중되지 않았는가?
- 부동층 이동 방향 예측에 근거 없는 정치적 전제가 없는가?

**④ 정책 평가 편향**
- 의제별 "승자" 판정에 이념적 기준이 개입되었는가?
- 실행 가능성 평가가 양 후보에게 동일하게 엄격하게 적용되었는가?

**⑤ 시나리오 편향**
- 시나리오 3종 중 특정 후보 유리 시나리오가 더 구체적이지 않은가?
- 리스크 분석이 한쪽에 집중되지 않았는가?

**산출물:**
- 편향 진단표: 5개 영역 × (현재 분석 / 편향 리스크 / 균형 대안)
- "제3자 눈 테스트": 두 캠프 모두 수용 가능한가?
- 캠프 밖 시선 체크리스트: 내부자 논리가 투영된 전제 목록
- 최종 판정: 균형 유지 / 부분 수정 / 전면 재검토
