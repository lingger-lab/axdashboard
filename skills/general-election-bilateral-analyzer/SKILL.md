```

선거: [선거연도] [선거구] [선거종류]

## 1단계: 핵심 전제 전환 (경선 → 본선)

```
[경선 프레임]          →    [본선 프레임]

당원 세그먼트              일반 유권자 세그먼트

## 2단계: 입력물 식별

### INSIGHTS 보고서 (여론분석)
- 식별어: "여론분석 보고서", "CampOne", "감성 분석", "인지도·지지도"
- 본선 활용:
  - 두 후보 지지율 갭 및 추이
  - 부동층 규모·성향·이동 방향
  - 정당 지지율 vs 후보 개인 지지율 괴리
  - 지역별·연령별 기반 분포
  - 이슈 TOP10 및 유권자 감성 분석

### POLICY 보고서 (전략분석)
- 식별어: "선거 전략 분석 보고서", "SWOT 분석", "승리 공식"
- 본선 활용:
  - 강·약점을 양자 비교 관점으로 재해석
  - 메시지 전략 → 유권자 전체 대상으로 재설계
  - 세그먼트 전략 → 본선 유권자 세그먼트로 전환

### 파일 미업로드 시
공개 정보 및 입력된 후보 정보만으로 분석을 진행한다.
분석 한계를 보고서 앞부분에 명시한다.

## 3단계: Part 1 — 양자 구도 핵심 분석

### [A. 구도 진단]

| 분석 항목 | 내용 |
|----------|------|
| 구도 유형 | 이념 대결형 / 인물 대결형 / 정책 대결형 / 복합형 |
| 지배 프레임 | 국정 심판론 / 지역 발전론 / 인물 경쟁론 |
| 부동층 규모 | 현재 추정치 + 최종 승패 영향도 |
| 지역 특수성 | 권역별 정당 기반 vs 후보 개인 매력도 |
| 핵심 변수 | 선거 결과를 가를 결정적 3개 변수 |

### [B. 여당 후보(A) 분석]

| 분석 항목 | 내용 |
|----------|------|
| 강점 | 집권당 자원 / 정부 정책 연계 / 조직망 |
| 약점 | 국정 지지율 연동 리스크 / 야당 프레임 피격 |
| 메시지 체계 | 3초 / 10초 / 30초 계층 |
| 표밭 전략 | 핵심 지지층 결집 + 캐스팅보트 공략 |
| 우위 시나리오 | 국정 프레임 유리 시 / 지역 이슈 부각 시 |

### [C. 야당 후보(B) 분석]

| 분석 항목 | 내용 |
|----------|------|
| 강점 | 심판론 활용 / 지역 밀착 차별화 |
| 약점 | 정책 실행력 검증 요구 / 조직 열세 가능성 |
| 메시지 체계 | 3초 / 10초 / 30초 계층 |
| 표밭 전략 | 진보 결집 + 중도 확장 우선순위 |
| 우위 시나리오 | 민심 이반 강할 때 / 인물론 부각 시 |

## 4단계: Part 2 — 확장 분석 섹션 (EXT-1~8)

### [EXT-1] 유권자 세그먼트 전략

```
입력: INSIGHTS 지역/연령/성향 데이터 + 후보 기반 분석
목표: 4~6개 유권자 세그먼트별 공략 설계

생성 항목:
1. 세그먼트 정의: 연령/지역/이념/생활권 기준
2. 세그먼트별 "현재 기울기 (A 우세/B 우세/중립)"
3. 세그먼트별 "전환 가능성 (高/中/低)"
4. 세그먼트별 "핵심 공략 채널"
5. 캐스팅보트 세그먼트 집중 분석 (규모 + 이동 조건)
6. 자원 배분 우선순위 매트릭스
```

### [EXT-2] 프레임 전쟁 시나리오

```
입력: 양 후보 메시지 + 정치 환경 변수
목표: 프레임 경쟁 시나리오 및 대응 설계

생성 항목:
1. 여당 후보가 원하는 프레임 vs 야당 후보가 원하는 프레임
2. 현재 지배 프레임 진단
3. 프레임 전환 시나리오 3종:
   - 시나리오①: 국정 심판론 강화 시 → 양 후보 대응
   - 시나리오②: 지역 이슈 부각 시 → 양 후보 대응
   - 시나리오③: 인물론 집중 시 → 양 후보 대응
4. 돌발 이슈 발생 시 프레임 전환 대응 매뉴얼
```

### [EXT-3] 정책 대결 분석

```
입력: 양 후보 공약 + INSIGHTS 이슈 TOP10
목표: 의제별 경쟁 우위 분석

생성 항목:
1. 10대 핵심 의제별 양 후보 공약 비교표
2. 차별점이 뚜렷한 의제 / 유사한 의제 분리
3. 의제별 "정책 승자" 판정 + 근거
4. 유권자 체감 정합성: 공약이 일상에 닿는가
5. 실행 가능성 검증: 권한/예산/절차 분류
6. 경쟁 우위 의제 TOP3 (각 후보별)
```

### [EXT-4] 메시지·카피 체계

```
입력: 양 후보 슬로건 + INSIGHTS 감성 분석
목표: 유권자 설득 메시지 경쟁력 비교

생성 항목:
1. 슬로건 경쟁력 비교 (기억성/감성/차별성)
2. 부동층 설득용 메시지 vs 지지층 결집용 메시지 분리
3. 세그먼트별 변형 메시지 (연령/지역/이념 맞춤)
4. TV 토론 예상 핵심 공방 3라운드 + 대응 템플릿
5. 위기 커뮤니케이션 즉답 스크립트 3종
6. 소셜미디어 카피 방향 (채널별)
```

### [EXT-5] 미디어·콘텐츠 전략

```
입력: 채널 현황 + INSIGHTS 데이터 수집 분석
목표: 채널별 경쟁 우위 및 콘텐츠 기획

생성 항목:
1. 채널별 경쟁 우위 비교 (TV/유튜브/인스타/지역언론/카카오)
2. 공식 선거운동 기간 기준 콘텐츠 캘린더
3. 리퍼포징 체계: 유세 1회 → 숏폼/카드/FAQ/블로그
4. AI 활용 가이드: 촬영→편집→자막→배포
5. 위기 커뮤니케이션 프로토콜 (네거티브 공격 시)
6. KPI: 도달률/공유율/완주율/여론 전환 지표
```

### [EXT-6] TV 토론·합동연설 전략

```
입력: 양 후보 강·약점 + 예상 쟁점
목표: 토론 승리 전략 및 답변 템플릿

생성 항목:
1. 예상 토론 주제 TOP5
2. 주제별 양 후보 공방 시나리오 (공격/방어 라인)
3. 각 후보의 "선제 공격 포인트" vs "방어 취약점"
4. 합동연설 3분 구조: 도입(30s)→문제(30s)→비전(60s)→근거(30s)→결의(30s)
5. 토론 후 콘텐츠 리퍼포징: 하이라이트→숏폼→카드→FAQ
6. 위기 답변 준비: 즉답이 필요한 예상 공격 5종
```

### [EXT-7] 선거 리스크 시나리오 관리

```
입력: 정치 환경 변수 + 양 후보 취약점
목표: 리스크별 대응 체계 수립

생성 항목:
1. 리스크 변수 8종 진단:
   - 돌발 스캔들 / 여론 급변 / 투표율 변동
   - 단일화 이슈 / 국정 사건 / 지역 재난
   - 제3후보 변수 / 중앙당 개입

2. 투표율 시나리오:
   - 고투표율(60%↑) → 유리한 후보 / 전략
   - 저투표율(45%↓) → 유리한 후보 / 전략

3. 승패 시나리오 3종:
   - A 우세 (격차 5%p↑): 조건 + 전략
   - 초접전 (격차 3%p 이내): 조건 + 전략
   - B 우세 (격차 5%p↑): 조건 + 전략

4. D-30~D-day 단계별 전략 조정 기준
```

### [EXT-8] 비판적 자기검증 — "이 분석이 공정한가?" ⚠️

**실행 시점: EXT-1~4 초안 완성 후, 최종 보고서 조립 전 반드시 실행**

```
목적: 분석자 편향 및 특정 후보 유리 편향을 식별하고 균형 확보

5대 검증 영역:

① 구도 진단 편향
   - 현재 진단이 특정 정당에 유리한 전제를 깔고 있는가?
   - 국정 심판론/지역 발전론 중 어느 쪽이 과다 반영되었는가?

② 후보 분석 형평성
   - A 후보와 B 후보 분석의 깊이·분량이 균형 잡혀 있는가?
   - 강점/약점 분석에서 한쪽에 유리한 프레임이 없는가?

③ 세그먼트 편향
   - 특정 세그먼트를 과대/과소 평가하고 있지 않은가?
   - 부동층 이동 방향 예측에 근거가 있는가?

④ 정책 평가 편향
   - 의제별 "승자" 판정에 이념적 기준이 개입되었는가?
   - 실행 가능성 평가 기준이 양 후보에게 동일하게 적용되었는가?

⑤ 시나리오 편향
   - 시나리오 3종 중 특정 후보 유리 시나리오가 더 구체적이지 않은가?
   - 리스크 분석이 한쪽에 집중되지 않았는가?

산출물:
- 편향 진단표: 5개 영역 × (현재 분석 / 편향 리스크 / 균형 대안)
- "제3자 눈 테스트": 두 후보 캠프 모두 이 분석을 수용할 수 있는가?
- 최종 판정: 균형 유지 / 부분 수정 / 전면 재검토
```

## 5단계: Part 3 — 선거 환경 변수

### [REF-1] 전국 정치 지형
- 현 정부 지지율 추이 + 지역 보정치
- 정당 지지율 vs 후보 개인 지지율 괴리 분석
- 여대야소/야대여소 구도가 지방선거에 미치는 영향

### [REF-2] 지역 특수 변수
- 해당 선거구 역대 선거 결과 패턴
- 핵심 지역 현안 (산업/개발/복지/교통 등)
- 유권자 인구 구조 변화 추이

### [REF-3] 투표율 분석
- 역대 동일 선거 투표율 추이
- 고/저 투표율 시 각 후보 유불리 시뮬레이션
- 투표율 관리 전략 비교

### [REF-4] 최종 승패 예측 변수
- 결정적 변수 TOP3 + 임계값
- 승패 분기점 D-30 / D-14 / D-7 점검 기준
