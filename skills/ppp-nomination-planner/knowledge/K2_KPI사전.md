# KPI 사전 정의 (축별)

사용자 상황에 맞게 조정하되, 아래를 기본 프레임으로 사용한다.

## 1. 당 기여도 KPI

| KPI | 측정 방법 | 증빙 예시 |
|-----|-----------|-----------|
| 공식 역할 수행 횟수/기간 | 당직·위원회 활동 로그 | 임명장, 회의록, 활동보고서 |
| 당 행사 참여/운영 횟수 | 참석 명단, 운영 기록 | 행사 참석확인서, 사진, 프로그램 |
| 정책 제안서/브리핑 작성 수 | 제출 문서 목록 | 정책제안서 사본, 접수확인 |
| 당내 TF 기여 건수 | TF 활동 기록 | TF 회의록, 결과보고서 |
| 선거 지원 활동 로그 건수 | 합법 범위 내 활동 기록 | 유세 참여 사진, 활동일지 |
| 교육/연수 참여 수료 | 수료증, 참석 기록 | 수료증 사본, 교육 수강 확인서 |

## 2. 지역 공적 활동 KPI

| KPI | 측정 방법 | 증빙 예시 |
|-----|-----------|-----------|
| 민원 접수→처리 완료 건수 | 민원 케이스 로그 | 민원 처리 대장, 결과 통보서 |
| 평균 민원 처리 리드타임 | 접수일~완료일 산술 | 민원 케이스 로그 날짜 기록 |
| 간담회/현장방문 횟수 | 활동 캘린더 | 간담회 사진, 참석자 명단, 회의록 |
| 간담회 참석자수 + 후속조치 완료율 | 명단 + 후속조치 로그 | 서명부, 후속조치 보고서 |
| 봉사시간/프로그램 횟수 | 봉사 기록 | 봉사확인서, 1365 기록 |
| 협약/연계기관 수 | 협약서 목록 | MOU, 협약서 사본 |

## 3. 도덕성 KPI

| KPI | 측정 방법 | 증빙 예시 |
|-----|-----------|-----------|
| 리스크 점검 완료율 | 체크리스트 항목 % | 자체 점검 체크리스트 |
| 논란 가능 발언/게시물 정비 완료율 | 점검 대상 대비 완료 수 | 점검 로그, 수정/삭제 기록 |
| 이해충돌 점검 결과 | 공개 가능 투명성 자료 마련 여부 | 이해충돌 자가 점검표 |
| 위기대응 문서 준비도 | FAQ/타임라인/입장문 준비 완료율 | 문서 파일 목록 |

## 4. 비교우위 KPI

| KPI | 측정 방법 | 증빙 예시 |
|-----|-----------|-----------|
| 차별화 3~5축 증빙 개수/품질점수 | 축별 증빙 수 + 품질 등급 | 증빙 레지스터 필터링 |
| 동일 축 반복성과(월별 지속성) | 월별 활동 건수 추이 | 월별 활동 로그 |

## KPI 채점 가이드라인

**정량 우선 원칙:** 모든 KPI는 가능한 한 숫자로 표현한다.

**채점 등급 기준 (자체 평가용):**
| 등급 | 기준 |
|------|------|
| A (상위 10%) | 동일 직위 지원자 평균 대비 150% 이상 |
| B (상위 30%) | 평균 대비 120~150% |
| C (평균) | 평균 대비 80~120% |
| D (보완 필요) | 평균 대비 80% 미만 |

**레이더 차트 차별화 전략:**
- 모든 축에서 B 이상을 목표로 하되
- 3~5개 축에서 A등급으로 '튀는 점'을 만든다
- 튀는 축은 반드시 실체·성과·증빙 중심으로 설계
