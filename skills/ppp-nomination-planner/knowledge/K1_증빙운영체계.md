## 파일명 규칙 (강제)

**형식:** `YYYYMMDD_평가요소_활동명_지역(또는기관)_증빙종류_v01`

**예시:**
- `20260410_지역공적_독거노인돌봄_○○동_봉사확인서_v01.pdf`
- `20260322_당기여_정책토론회_시당_참석확인서_v01.pdf`
- `20260415_도덕성_이해충돌점검_자가_체크리스트_v01.xlsx`

**버전 관리:**
- v01 → v02로 증가
- 최종본은 FINAL 추가 가능: `..._v03_FINAL.pdf`
- 사진/영상: 원본 보관 + 사용본(크롭/자막) 분리 저장

## 증빙 레지스터 스키마

각 증빙을 1줄로 기록한다 (스프레드시트/노션 권장):

| 컬럼 | 설명 | 예시 |
|------|------|------|
| ID | 고유번호 | EVD-001 |
| 날짜 | 활동 날짜 | 2026-04-10 |
| 평가요소 | 당기여/지역공적/도덕성/비교우위 | 지역공적 |
| 활동명 | 구체적 활동명 | 독거노인 돌봄 봉사 |
| 장소/기관 | 활동 장소 또는 기관 | ○○동 주민센터 |
| 참여자수 | 참여 인원 | 15명 |
| 성과(정량) | 최소 1개 이상 숫자 | 방문 12가구, 4시간 |
| 담당자 | 내부 담당자 | 홍길동 |
| 증빙유형 | 문서 종류 | 봉사확인서 |
| 파일경로 | 폴더 내 위치 | 02_지역공적/02-1_봉사·나눔/ |
| 확인자/발급처 | 증빙 발급 기관/인 | ○○동 주민센터장 |
| 공개가능 | Yes/No | Yes |
| 리스크 | 리스크 메모 | 없음 |
| 비고 | 추가 사항 | 언론 보도 연계 가능 |

**성과(정량) 예시:**
- 해결 건수, 참석자 수, 봉사 시간
- 협약 수, 민원 처리기간 단축일
- 예산 절감액, 수혜자 규모

## 증빙 품질 체크리스트

모든 증빙은 아래 4항목을 통과해야 한다:

1. **5W1H + 결과 명확성**: 누가(실명/직함) / 언제 / 어디서 / 무엇을 / 얼마나(숫자) / 결과가 명확한가?
2. **검증 가능성**: 발급처·확인자·연락경로가 합법적/공식적 범위에서 검증 가능한가?
3. **사실 중심**: 과장 표현 없이 사실 중심인가?
4. **개인정보 최소화**: 개인정보/민감정보는 최소화했는가?

## 증빙 확보 우선순위 매트릭스

| 우선순위 | 기준 | 예시 |
|---------|------|------|
| 1순위 | 공식 기관 발급 + 정량 성과 | 봉사확인서, 감사장, 공식 참석명단 |
| 2순위 | 공식 기관 발급 + 정성 성과 | 협력기관 추천서, 공식 회의록 |
| 3순위 | 자체 기록 + 외부 확인 가능 | 활동 사진(날짜/장소 메타데이터) + 참석자 서명 |
| 4순위 | 자체 기록만 | 내부 활동일지 (보완 필요) |
