# 채널별 이미지 프롬프트 가이드

## 공통 비주얼 아이덴티티

### Brad Kim 비주얼 세계관
- **핵심 상징:** 콘크리트 틈의 민들레, 새벽 빛, 모니터 불빛, 식은 커피
- **인물 묘사:** 중년 남성 뒷모습/실루엣/손 클로즈업 (얼굴 없음)
- **공간:** 새벽 아파트 서재, 베란다, 카페 창가
- **시간:** 새벽 4~5시 (동트기 전 남색+오렌지 그라데이션)
- **색감 팔레트:**
  - 주색: 네이비(#1a1a2e), 웜그레이(#b8b0a0)
  - 강조: 새벽 오렌지(#e8a87c), 모니터 블루(#4fc3f7)
  - 배경: 웜 화이트(#f5f0e8), 소프트 다크(#2d2d3a)
- **질감:** 수채화풍, 미니멀, 약간의 그레인/노이즈

### 절대 포함하지 않는 것
- 얼굴 정면 (프라이버시 + 보편성)
- 화려한 테크 이미지 (로봇, 회로판, 홀로그램)
- 성공/부의 상징 (고급차, 빌딩, 돈)
- 과도한 텍스트 (카카오 카드뉴스 제외)
- 클립아트/스톡포토 느낌

## 채널별 이미지 스펙

### 1. 브런치 (2장)

**커버 이미지 (16:9 / 1200×675px)**
```
[프롬프트 템플릿]
스타일: 수채화, 따뜻한 톤, 미니멀
장면: [에세이 핵심 장면에서 파생]
  예) 새벽 창가 + 노트북 + 식은 커피
톤: 고요하면서 따뜻한
색감: 네이비(#1a1a2e) + 새벽 오렌지(#e8a87c) + 웜그레이(#b8b0a0)
구도: 중앙 또는 1/3 법칙
인물: 뒷모습 실루엣 (있을 경우)
텍스트: 없음
용도: 브런치 커버 — 피드에서 스크롤을 멈추게 하는 이미지
```

**본문 삽입 이미지 (4:3 / 1200×900px)**
```
[프롬프트 템플릿]
스타일: 미니멀 수채화 또는 라인드로잉
장면: [에세이 전환점의 상징적 이미지]
  예) 콘크리트 틈 민들레, 키보드 위의 손, 메모장
톤: 에세이 감정에 맞춤 (고요/긴장/따뜻)
색감: 메인 색상 2개 + 배경
텍스트: 없음
용도: 본문 중간 삽입 — 독자의 시선을 환기
```

### 2. 네이버 블로그 (3장)

**대표 이미지 (16:9 / 1200×675px)**
```
[프롬프트 템플릿]
스타일: 깔끔한 인포그래픽풍, 미니멀
장면: [글의 주제를 시각적으로 요약]
  예) 노트북 화면에 코드가 보이는 서재 풍경 (정돈된 느낌)
톤: 전문적이면서 접근 가능한
색감: 네이비 + 화이트 + 포인트 블루
구도: 대칭 또는 깔끔한 구성
인물: 선택적 (있다면 뒷모습)
텍스트: 없음
용도: 네이버 검색 썸네일 + 블로그 대표 이미지
특이사항: 브런치보다 "정보적" 느낌. 수채화보다는 사진풍/일러스트풍.
```

**본문 이미지 (4:3 / 1200×900px) ×2**
```
[프롬프트 템플릿 — 단계/과정 설명용]
스타일: 미니멀 다이어그램풍 또는 스크린샷풍
장면: [글에서 설명하는 과정/단계의 시각화]
  예) 프롬프트 작성 과정, before/after 비교
톤: 명확하고 교육적인
색감: 밝은 배경 + 강조색 1개
텍스트: 최소한의 라벨 (있을 경우 한국어)
용도: 설명 보조 — 단계별 과정을 시각화
```

### 3. 카카오 채널 (1장 — 카드뉴스)

**카드뉴스 이미지 (1:1 / 1080×1080px)**
```
[프롬프트 템플릿]
스타일: 미니멀 + 텍스트 오버레이 공간 확보
장면: [감성적 배경 — 텍스트가 잘 읽히는 단순한 구성]
  예) 새벽 하늘 그라데이션, 창가 실루엣, 커피잔 클로즈업
톤: 감성적, 따뜻한
색감: 어두운 배경(네이비/다크그레이) + 따뜻한 하이라이트
구도: 상단 2/3 배경 + 하단 1/3 여백 (텍스트 영역)
텍스트: 포함 — 아래 텍스트 오버레이 스펙 참조
```

**텍스트 오버레이 스펙:**
```
위치: 이미지 중앙 또는 하단 1/3
폰트: Pretendard 또는 Noto Sans KR
크기: 본문 24~28pt / 워터마크 10pt
색상: 흰색 (#FFFFFF)
배경: 반투명 블랙 오버레이 (rgba(0,0,0,0.5)) — 텍스트 영역에만
내용: 에세이의 통찰 문장 1개 (15자 이내 × 2~3줄)
하단 워터마크: "새벽메모 by Brad Kim" (우측 하단, 작고 은은하게)
```

**카드뉴스 텍스트 예시:**
```
"프롬프트가
결국 기획서였다"

— 새벽메모 by Brad Kim
```

### 4. 페이스북 (1장)

**링크 프리뷰 이미지 (1.91:1 / 1200×628px)**
```
[프롬프트 템플릿]
스타일: 감성 사진풍, 따뜻한 톤
장면: [에세이의 감정을 전달하는 넓은 풍경/공간]
  예) 새벽 도시 풍경, 서재 전경, 카페 창밖
톤: 영화 스틸컷 같은 (시네마틱)
색감: 약간 데새추레이션 + 따뜻한 톤
구도: 가로로 넓은 구성 (1.91:1 비율에 최적화)
인물: 뒷모습 실루엣 (작게, 풍경 속에)
텍스트: 없음 (페이스북이 OG 태그에서 제목을 가져옴)
용도: 페이스북 뉴스피드에서 링크 공유 시 프리뷰 이미지
특이사항: 좌우 여백 넉넉하게 — 모바일에서 잘려도 핵심 안 잘리게
```

## 씨앗 기반 이미지 커스터마이징

사용자가 제공한 인간 씨앗에 시각 요소가 있으면 이미지에 반영합니다:

| 씨앗 내용 | 이미지 반영 |
|-----------|------------|
| "비가 온다" | 창문에 빗방울, 젖은 베란다 |
| "카페에서 봤다" | 카페 내부 풍경, 노트북 |
| "동현이와 대화" | 두 사람의 뒷모습/실루엣 |
| "에러 메시지" | 모니터 화면 클로즈업 (텍스트는 흐릿하게) |
| "새벽 산책" | 어두운 골목 + 동트는 하늘 |
| "커피" | 커피잔 클로즈업, 김이 올라오는 |

씨앗에 시각 요소가 없으면 에세이의 감정 톤에 맞는 기본 이미지를 생성합니다.

## 이미지 생성 도구 가이드

### 추천 도구 (우선순위)
1. **Midjourney** — 수채화/감성 이미지 최적
2. **DALL-E 3** — 텍스트 오버레이 정확도 높음 (카드뉴스)
3. **Stable Diffusion** — 일관된 스타일 유지 가능
4. **Canva** — 카드뉴스 텍스트 오버레이 후작업

### 프롬프트 작성 언어
- 모든 이미지 프롬프트는 **영어**로 작성 (생성 품질 최적화)
- 한국어 설명은 참고용으로 병기

### 일관성 유지를 위한 공통 프롬프트 접미사
```
모든 이미지 프롬프트 끝에 추가:
"--style: watercolor minimalist, warm tones, 
  color palette: navy #1a1a2e, warm gray #b8b0a0, dawn orange #e8a87c,
  no text, no faces, subtle grain texture, 
  aspect ratio: [채널별 비율]"
```

네이버용만 예외:
```
"--style: clean editorial, professional, slightly warm,
  color palette: navy, white, accent blue #4fc3f7,
  no text, no faces,
  aspect ratio: [비율]"
```
