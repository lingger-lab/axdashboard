# 퍼널 & CTA 전략

## 전체 퍼널 구조

```
브런치 에세이 (인지 + 공감)
    ↓ 구독 (정기 독자화)
    ↓ 프로필 클릭 → 소개글에서 세계관 확인
    ↓ CTA → 오픈채팅방 (관계 형성)
    ↓ 오픈채팅 내 자연스러운 대화
    ↓ "이런 것도 도와드릴 수 있어요" (가치 제시)
    ↓ 예약폼/상담 신청
    ↓ 유료 서비스 (워크숍/컨설팅/바이브코딩 교육)
```

핵심: **모든 단계가 자연스럽고, 어느 단계에서도 "팔고 있다"는 느낌이 없어야 합니다.**

## CTA 유형별 가이드

### 유형 1: 동료형 CTA
**톤:** "같이 고민하는 사람들이 있다"
**적합한 에세이:** 불안, 고민, 전환기 주제

```
예시:
"이런 고민, 혼자 하고 계신 건 아니죠?
비슷한 길을 걷는 분들과 새벽 메모를 나누고 있습니다.
→ [새벽메모 오픈채팅]"

"AI 시대, 뭐부터 해야 할지 모르겠다면.
저도 그랬습니다. 같이 흔들리는 사람들이 여기 있어요.
→ [오픈채팅 참여하기]"
```

### 유형 2: 후속편형 CTA
**톤:** "이 이야기의 뒷이야기가 있다"
**적합한 에세이:** 시행착오형, 연재물

```
예시:
"이 다음에 무슨 일이 있었는지,
오픈채팅에서 먼저 이야기했습니다.
→ [채팅방에서 뒷이야기 보기]"

"이 실패담의 후속편이 궁금하시면.
→ [다음 이야기를 먼저 받아보기]"
```

### 유형 3: 자원공유형 CTA
**톤:** "내가 쓰는 도구/방법을 나눈다"
**적합한 에세이:** 통찰형, 바이브코딩 관련

```
예시:
"글에서 언급한 프롬프트 템플릿,
채팅방에서 공유하고 있습니다.
→ [템플릿 받으러 가기]"

"바이브코딩 시작 가이드,
같이 해보는 분들께 나누고 있어요.
→ [오픈채팅]"
```

### 유형 4: 질문형 CTA
**톤:** "당신의 이야기를 듣고 싶다"
**적합한 에세이:** AI철학형, 뉴스반응형

```
예시:
"이 질문, 당신은 어떻게 답하시겠어요?
같이 고민하는 공간이 있습니다.
→ [대화에 참여하기]"

"혼자 생각하기엔 무거운 질문이라면.
→ [여기서 같이 이야기해요]"
```

### 유형 5: 무반응형 (CTA 없음)
**적합한 에세이:** 짧은몰입형, 감정 순도가 높은 글

CTA가 글의 여운을 깨뜨릴 때는 넣지 않습니다.
3편 중 1편은 CTA 없이 순수 에세이로 남깁니다.
→ 이것이 오히려 신뢰를 만듭니다.

## CTA 배치 규칙

| 규칙 | 설명 |
|------|------|
| 위치 | 에세이 본문이 완전히 끝난 후. 한 줄 빈 줄 뒤 |
| 길이 | 2~3줄 이내 |
| 톤 | 에세이 톤과 동일. 갑자기 마케팅 톤으로 바꾸지 않음 |
| 빈도 | 3편 중 2편에만 삽입. 매번 넣지 않음 |
| 링크 | 오픈채팅 링크만. 직접 상품/예약폼 연결 안 함 |
| 금지 | "지금 바로!", "한정!", "무료!", "놓치지 마세요" |

## 오픈채팅방 운영 전략

### 채팅방 이름 후보
- "새벽메모 — AI 시대를 걷는 사람들"
- "늦은 출발 모임"
- "콘크리트 민들레 카페"

### 채팅방 내 전환 설계
```
1주차: 환영 + 자기소개 유도
2주차: 브런치 글 관련 대화 유도
3주차: "이런 고민이 있으시면 도움 드릴 수 있어요" 자연 언급
4주차~: 워크숍/상담 안내 (요청한 사람에게만)
```

### 채팅방 → 예약폼 전환 문구 (자연스러운 버전)

```
"여러 분들이 비슷한 질문을 해주셔서,
한번 정리해서 나눠드리려고 합니다.
소규모로 진행하는 워크숍 관심 있으시면 여기에 →
[예약폼 링크]"

"1:1로 이야기 나누고 싶으신 분,
편하게 예약 잡아주세요.
부담 없는 30분 대화입니다.
→ [상담 예약]"
```

---

## 전환율 최적화 원칙

1. **느린 전환이 강한 전환이다**
   - 첫 글에서 바로 오픈채팅으로 보내지 않음
   - 3~5편의 글로 신뢰를 쌓은 뒤 CTA 시작

2. **팔지 않는 사람이 가장 잘 판다**
   - 에세이의 진정성이 전환의 엔진
   - CTA는 "이 사람과 더 이야기하고 싶다"는 자발적 욕구에서 작동

3. **오픈채팅은 중간역이다**
   - 브런치 → 오픈채팅 → 예약폼
   - 오픈채팅을 건너뛰고 바로 예약폼으로 가면 전환율 급락

4. **CTA 없는 글이 CTA 있는 글을 도와준다**
   - 매번 CTA를 넣으면 "광고 블로그"가 됨
   - CTA 없는 순수 에세이가 브랜드 신뢰를 축적

## 예약폼 설계 가이드

### 예약폼 포함 항목
- 이름 (선택)
- 연락처
- "어떤 고민이 있으신가요?" (자유 기술)
- 선호 시간대
- "Brad의 글 중 어떤 글이 마음에 남았나요?" (선택)
  → 이 질문이 관계의 온도를 높임

### 예약폼 상단 문구
```
"이 폼을 작성하신다고 바로 무언가를 사셔야 하는 건 아닙니다.
편하게 30분 대화를 나눕니다.
AI 시대, 뭐부터 시작해야 할지 같이 정리해보는 시간입니다."
```
