# AX 전환 서사기 (스킬 4 · ② 생성 > 전환)

Brad의 자기 서사로 잠재고객의 **전환**을 만드는 스킬. 퍼널 최하단 — 정보(인식)도
검색노출(발견)도 아닌, "이 사람이라면 믿겠다"는 정서를 만든다.

## 경계

```
[스킬 1~3]    정보·신호·검색노출
        ↓
[스킬 4: 에세이]  Brad 서사 → 정서 신뢰 → 30분 1:1 전환  ← 지금 이 스킬
```

**입력:** Brad의 인간 씨앗(오늘의 한 문장/감정/장면)
**출력:** 브런치/페북용 1인칭 에세이 + 채널별 이미지 프롬프트

## 핵심: Brad의 서사가 AX전환의 산 증거다

이 스킬은 가장 적게 바뀐다. Brad의 원점 스토리(Denpura: 온라인강의 헛돈 → 자막생성기
3개월 5번 실패 6번째 성공 → 지수적 가속)가 이미 AX전환 그 자체이기 때문이다.
추가하는 건 그 서사를 **고객의 전환 여정에 포개는 다리 한 줄.**

## AX 격상 포인트 (이 스킬에 추가되는 것)

- 에세이 말미에 **전환 다리**: "내가 겪은 이 전환을, 당신 회사에도"
- 4기둥 중 그 에세이가 닿는 **1개 기둥으로 착지** (보통 인건비 or 수익모델)
- CTA = "AX 코칭"이 아니라 **"30분 1:1"**(오픈채팅). 개념어로 끝내지 않는다.

> Brad의 무기는 개념이 아니라 경험이다. "AX 컨설팅"의 경쟁자는 삼정KPMG지만,
> "직접 해본 59세 사장"의 경쟁자는 없다. 서사로 승부한다.

## 워크플로우 (기존 brunch 구조 유지 + AX 다리)

```
[Phase 0] 인간 씨앗 요청    — 오늘의 한 문장/감정/장면
[Phase 1] 당일 뉴스 검색    — 씨앗과 닿는 AI/경제/사회 뉴스
[Phase 2] 서사 직조         — 씨앗 + Brad 원점 + 뉴스
[Phase 3] AX 다리 놓기      — 4기둥 1개로 착지 (★ 추가된 부분)
[Phase 4] AI냄새 제거 검증
[Phase 5] 출력             — 에세이 + 이미지 프롬프트 + CTA
```

## Phase 0: 인간 씨앗 요청

```
오늘의 씨앗을 주세요. 한 문장, 한 감정, 한 장면이면 됩니다.
- 오늘 아침 떠오른 생각
- 누군가와의 대화 한 토막
- 문득 든 후회나 기대
나머지는 제가 키웁니다.
```

## Phase 3: AX 다리 (격상의 실체)

서사가 정서적 절정에 닿은 뒤, *억지 영업 없이* 다리 한 줄을 놓는다.
예: "그때 내가 혼자 끙끙댄 그 일을, 지금은 AI가 한다. 당신 회사의 그 일도 그럴 수 있다."
→ 자연스럽게 인건비/수익모델 기둥으로 착지 → 30분 1:1 CTA.

## ⚠️ 검증 단계에서는 보류

이 스킬(전환)은 인용 증거가 나온 *뒤* 합류시킨다. 최소 검증 루프는 큐레이터+블로그(AEO)
2개면 충분하다. 에세이를 먼저 세우면 검증 전 운영 부담만 는다.
