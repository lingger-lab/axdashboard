# AX 신호 수집기 (스킬 1 · ① 공급)

집출(zipchul.cloud) 콘텐츠 파이프라인의 **입력단**. 세 생성 스킬(뉴스레터·블로그·에세이) 모두의 공통 원료를 공급한다.

## 이 스킬의 정확한 범위 (경계가 핵심)

```
[스킬 1: 큐레이터]  신호 수집 + 4기둥 태깅 → Brad에게 후보 제시   ← 지금 이 스킬
        ↓ Brad가 1건 선택 + 코멘트(씨앗) 입력
[스킬 2~4: 생성]    선택된 1건 → 뉴스레터/블로그/에세이
        ↓
[스킬 5: 숏폼]      완성된 글 → 숏폼 증폭
```

**입력:** 없음 (Brad가 "오늘 신호" 요청 시 능동 수집) 또는 Brad가 던진 키워드
**출력:** 4기둥별로 분류된 신호 후보 리스트 (건당: 제목·출처·기둥태그·왜 걸리는지 한 줄)
**자동화 경계:** 수집·태깅 = 자동 / **최종 선택 = 사람(Brad의 안목 = 차별점)**

## 핵심 원칙: AX 4기둥이 수집의 그물이다

모든 신호는 아래 4기둥 중 **정확히 하나**에 걸릴 때만 후보가 된다. 안 걸리면 버린다.

| 기둥 | 고객 언어 (이게 콘텐츠 표면) | 걸리는 신호 예시 |
|---|---|---|
| 업무자동화 | "이 반복일, 안 해도 되게" | 자동화 툴 출시, 반복업무 AI 대체 사례 |
| 생산성향상 | "한 명이 세 명 몫을" | AI 협업 사례, 1인 생산성 도구 |
| 인건비절감 | "직원 안 뽑고 버티기" | 최저임금·인력난·채용난·폐업률 |
| 수익모델개선 | "같은 일로 더 버는 구조" | 신규 비즈모델, 1인 사업 성공 사례 |

> ⚠️ "AX전환"이라는 개념어는 후보 리스트에 **쓰지 않는다.** 태그는 Brad 백엔드용이고,
> 콘텐츠 표면에 나갈 건 항상 오른쪽 칸(고객 언어)이다.

## 지역 가중치 (Brad 타깃 정렬)

경남 4050 사장님이 타깃이므로, 같은 기둥이면 **지역 신호에 가중치**를 둔다:
창원(기계·방산) · 거제(조선) · 김해(금형) · 경남(농업·관광) 관련 신호 우선.

## 발행 철학: 매일 수집, 주 3회 발행

큐레이터는 **매일** 후보를 던지되, 발행은 Brad가 고른 **주 3회**만. 논평할 가치가 있는
신호는 매일 나오지 않는다. 침묵할 권리가 신뢰다. 큐레이터의 일은 "발행 강제"가 아니라
"고를 거리를 좁혀주는 것".

## 워크플로우

```
[Phase 0] 수집 범위 확인   — "오늘 신호" or Brad 키워드
[Phase 1] 신호 수집        — web_search로 4기둥 관련 뉴스 검색
[Phase 2] 4기둥 태깅        — 각 신호를 기둥 1개에 매핑 (안 걸리면 버림)
[Phase 3] 지역 가중치 적용  — 경남 신호 상단 배치
[Phase 4] 후보 제시        — 기둥별 분류 리스트 + 선택 유도
```

## Phase 1: 신호 수집

검증 단계에서는 **인건비절감 기둥 1개에 집중**한다(가장 즉각적 통증). 4기둥 전체 수집은
인용 증거가 나온 뒤 확장. 검색어 예: "소상공인 인력난", "최저임금 2026", "1인 자영업 AI".

## Phase 2: 4기둥 태깅

각 신호에 대해 물어본다: "이 뉴스가 사장님의 어떤 통증을 건드리나?" 답이 4기둥 중 하나로
딱 떨어지지 않으면 버린다. 억지로 끼우면 입구가 오염되고 위에 쌓는 콘텐츠가 다 어긋난다.

> ⚠️ 검증 초기에는 태깅도 Brad가 손으로 확인. 자동 태깅은 패턴이 쌓인 뒤 켠다.

## Phase 4: 후보 제시 (출력 형식)

```
오늘의 신호 — [날짜]

🔴 인건비절감 ("직원 안 뽑고 버티기")
  1. [제목] / [출처]
     → 왜 걸리나: [한 줄]
  2. ...

🟡 업무자동화 ("이 반복일, 안 해도 되게")
  ...

고르실 1건과 한마디(씨앗)를 주세요. 나머지는 생성 스킬이 키웁니다.
```

## 다음 스킬로 넘기기

Brad가 1건 + 코멘트를 정하면 → 의도에 따라 분기:
- 시의성 있는 해석 → **스킬 2 (뉴스레터)**
- AI 검색 노출 목적 → **스킬 3 (블로그/AEO)**
- 정서적 신뢰 구축 → **스킬 4 (에세이)**
한 신호가 셋 모두로 갈 수도 있다 (같은 원료, 세 온도).
