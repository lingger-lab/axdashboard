# 보완·확장 워크플로우 가이드 v4.0 (일반화)

> POLICY 보완·확장 보고서 생성의 단계별 실행 가이드.
> 모든 단계에서 **{선거모드} 승리**가 최우선 목표.
> `{플레이스홀더}`는 STEP 0에서 수집된 파라미터로 자동 치환된다.

---

## STEP 0 — 파라미터 수집 (작업 시작 전 필수)

SKILL.md의 "STEP 0 — 파라미터 수집" 절차를 먼저 실행한다.
5개 항목이 확정되면 아래 단계를 진행한다.

```
확정 파라미터:
- {후보자명}:
- {지역명}:
- {선거종류}:
- {선거모드}: [경선 / 본선 / 통합]
- {경선방식}:
- {경쟁자1}~{경쟁자N}:
- {권역1}/{권역2}/{권역3}:
- {선거일}: D-day =
```

## 1단계: 입력물 읽기 및 식별

```
1. pandoc으로 업로드 파일 텍스트 추출
2. 보고서 종류 식별:
   - "여론분석 보고서" / "CampOne" → INSIGHTS
   - "선거 전략 분석 보고서" / "SWOT 분석" → POLICY
3. 두 보고서 동시 업로드 시 → 통합 보완 모드
4. 파일 없을 시 → 파라미터만으로 생성 (신속 모드)

[선거모드 분기 시작]
- 경선 모드  → Part 1 경선 중심, Part 3 본선 참고
- 본선 모드  → Part 1 본선 중심, 경선 내용 축소
- 통합 모드  → Part 1 경선 우선 + Part 2 후반 본선 전환 포함
```

## 2단계: 기존 POLICY 내용 파악

### ME 파악 — "{후보자명}을/를 왜 뽑아야 하나?"
- 프로필에서 **{선거모드} 스토리** 핵심 추출
- 강점 중 **경쟁자 대비 우위** 항목 식별
- 약점 중 **{선거모드}에서 공격당할** 항목 식별
- SWOT 전략 중 **실행 가능한** 항목과 **추상적** 항목 분리
- 포지셔닝에서 **{경쟁자N}인 비교 축** 확인

### FIELD 파악 — "경쟁자 누구를 어떻게 이기나?"
- 경쟁자 {N}인의 **인지도/지지도 순위** 추출
- 경쟁자별 **핵심 강점 1개 + 약점 1개** 추출
- 전략적 권고의 **대응 프레임** 추출
- {지역명} 현안에서 **유권자가 관심 있는 의제** 식별
- PEST에서 **{선거종류}에 영향 줄 정치 변수** 추출

### PLAN 파악 — "{경선방식} 이중전략의 구체성은?"
- 표밭 세그먼트의 **규모/현재지지율/목표** 추출
- 메시지 전략의 **슬로건/세그먼트별 메시지** 추출
- 로드맵 4단계의 **마일스톤과 임계경로** 추출
- 리스크 대응의 **{선거종류} 특유 리스크** 식별

### DO 파악 — "실행 설계가 유권자 공략에 맞춰져 있나?"
- 세그먼트별 정책수립의 **문제/해결/기대효과** 구조 추출
- 콘텐츠 방향의 **채널별 가이드** 추출
- 바이럴 아이디어 **3종** 추출
- 실행 가이드라인의 **발표 전략과 리스크 완화** 추출

### 부록 파악
- 슬로건 옵션과 **추천 옵션** 추출
- 핵심 공약의 **내용과 타깃** 추출
- 핵심 정책 목록의 **키워드** 추출

## 3단계: Part 1 — POLICY 보완 답변 생성

### 생성 포맷

```
■ [POLICY 섹션명] 보완

[기존 내용]
POLICY에서 {요약}을 제시하고 있다.

[보완 관점] [QA-Q{번호}]
{질문}에 따르면, {보완 포인트}를 추가하면 {선거모드}에서 더 강해진다.

[보완 답변]
{POLICY 기존 내용}과 {INSIGHTS 데이터}를 참조하여:
{구체적 보완 내용 — 즉시 사용 가능한 산출물}

[근거] [POLICY-{섹션}], [INSIGHTS-섹션{번호}], [QA-Q{번호}]
```

### ME 보완 생성 가이드

| 보완 항목 | Q 적용 | 산출물 |
|---------|--------|------|
| 시그니처 스토리 | Q1 | 10초/30초/120초 3종 |
| 경쟁자 대비 강점 | Q2, Q-N2 | "{경쟁자1}와 비교하면..." 활용 시나리오 |
| 약점 프레임 전환 | Q9, Q-N2 | 경선/본선 방어 답변 |
| 3초 차별화 | Q11 | {선거종류} 후보 중 유일한 것 한 문장 |
| SWOT→전략 | Q8 | "D-XX에 이것 실행" |
| 포지셔닝맵 | Q2 | {경쟁자N}인을 2축에 배치 |

### FIELD 보완 생성 가이드

| 보완 항목 | Q 적용 | 산출물 |
|---------|--------|------|
| 유권자 관심 의제맵 | Q3, Q10 | {권역1}/{권역2}/{권역3} 유권자 의제 TOP3 |
| 🆕 경쟁자 1:1 대응표 | **Q-N2** | {경쟁자N}인 × (한문장대비/공격방어TOP3/표밭겹침/대응우선순위) |
| 🆕 1위 양자대결 분석 | **Q-N8** | {경쟁자1} 양자구도 시: 표이동 시뮬레이션/프레임전환/시나리오3종 |
| 경쟁자 대응 프레임 확장 | Q-N2 | 1줄 요약 → 상황별 시나리오 |
| PEST 선거 변수 | Q7 | 당내역학/공천룰/중앙당 변수 |

### PLAN 보완 생성 가이드

| 보완 항목 | Q 적용 | 산출물 |
|---------|--------|------|
| {경선방식} 수치화 | Q1 | 핵심유권자 목표표수 + 여론조사 목표점수 분리 |
| 🆕 표밭 전환 전술표 | **Q-N1** | 세그먼트 × (미지지이유3/전환트리거/접점×빈도×메시지) |
| 이중전략 매트릭스 | Q5 | 주간 병렬 실행표 |
| 선거 리스크 시나리오 | Q6, Q7 | {선거종류} 특유 리스크 3종 시나리오 |
| 메시지 3층 체계 | Q11 | 3초/10초/30초 × 핵심유권자용/여론조사용 |
| 🆕 4단계 병렬 마일스톤 | **Q-N4** | 단계별 분리 + 주차별 핵심1개 |

### DO 보완 생성 가이드

| 보완 항목 | Q 적용 | 산출물 |
|---------|--------|------|
| 유권자 유형별 공략 | Q10, Q-N1 | 세그먼트 × 연령/지역/계층 하위분류 |
| 🆕 3자 정합표 | **Q-N3** | 표밭 × 공약 × 메시지 매트릭스 + 불일치 보완 |
| 채널 매트릭스 | Q12 | 오프라인 접점 + 디지털 역할분담 |
| 바이럴 보완 | Q12 | 기존 3종 + {선거종류} 특화 추가 |
| 🆕 조직 운영 | **Q-N6** | DB/간담회/전화문자/자원봉사 매뉴얼 |

## 4단계: Part 2 — 확장 섹션 생성 (EXT-1~7)

### EXT-1 개인 브랜딩

```
입력: POLICY-ME + INSIGHTS SWOT/감성
{선거모드} 관점: "유권자가 봤을 때 매력적인" 브랜딩

생성:
1. 브랜드 정체성 (What/How/Why)
2. 시그니처 스토리 3종
3. 키워드 5개 + 금지 키워드 5개
4. 신뢰 자산 목록 (유권자가 검증 가능한)
5. 경쟁자 {N}인 브랜드 비교표
```

### EXT-2 이미지메이킹

```
입력: POLICY-ME 프로필 + 차별화
{선거모드} 관점: 유권자 접점에서의 이미지

생성:
1. 비주얼 아이덴티티 (컬러/폰트/사진톤)
2. 언어 아이덴티티 (말투/선호어/금지어)
3. 상황별 이미지: 간담회/현장/미디어/SNS
4. 촬영 가이드 (앵글/배경/소품)
```

### EXT-3 브랜딩 카피

```
입력: POLICY 슬로건 + 핵심 공약 + INSIGHTS 이슈/감성
{선거모드} 관점: 핵심 유권자 대상 = 메인, 시민 대상 = 여론조사용

생성:
1. 메인 슬로건 최종 추천
2. 서브 카피: 세그먼트별 변형
3. 공약별 원라이너 N종
4. 유권자 FAQ 답변 TOP10 (핵심3줄+근거)
5. 경선 경쟁자 공격 대응 카피 (경쟁자별)
6. 소셜미디어 카피
7. 여론조사 대비 시민 메시지 (별도 세트)
```

### EXT-4 공약 구체화

```
입력: POLICY 핵심 공약 + INSIGHTS 이슈 TOP10
{선거모드} 관점: "유권자가 납득하는" 실행가능성

생성:
1. 가능/검토/불가 분류표
2. 공약별 실행 카드 (배경/목표/예산/권한/절차/일정/KPI)
3. Before→After 대비표
4. 경쟁자 {N}인 공약 대비표
5. 시민용 검증표 1장
```

### EXT-5 콘텐츠 기획

```
입력: POLICY-DO 콘텐츠 + INSIGHTS 데이터수집
{선거모드} 관점: 핵심 유권자 채널 최우선 + 디지털 병행

생성:
1. 채널 매트릭스 (오프라인 접점 + 디지털)
2. 콘텐츠 유형 5종 가이드
3. 리퍼포징 체계 (간담회→숏폼/카드/FAQ)
4. 주간 콘텐츠 캘린더
5. 공약별 콘텐츠 시리즈 기획
6. CIVIC HUB 설계
7. 현장-디지털 싱크 루프
8. AI 활용 가이드
```

### EXT-6 토론/합동연설회 🆕

```
입력: POLICY-FIELD 경쟁자 분석 + ME 약점
적용: Q-N5, Q9, Q-N2

생성:
1. 예상 토론 주제 TOP5
2. 주제별 답변 템플릿 (팩트→원칙→조치→시간표)
3. 경쟁자별 토론 대비
4. 합동연설회 3분 핵심 연설 구조
5. 토론 후 콘텐츠 리퍼포징
6. 위기 답변 준비 ({후보자_리스크} 등)

[선거모드 분기]
경선: 합동연설회·경선 토론 중심
본선: 공중파 토론·정책 토론 중심으로 교체
```

### EXT-7 조직 운영 🆕

```
입력: POLICY-PLAN 핵심전략 + DO 실행가이드라인
적용: Q-N6, Q5, Q-N1

생성:
1. 유권자 DB 관리 (수집→세그먼트→접촉이력→관리)
2. 간담회 매뉴얼 (의제→진행→Q&A→후속)
3. 전화·문자 스크립트 3종 + 타이밍
4. 자원봉사자 관리 (단계별 확대)
5. 접점 주3회 실행표
6. 유권자 불만 TOP5 + 고정 답변

[선거모드 분기]
경선: 당원 DB + 당원 접점 중심
본선: 일반 시민 자원봉사자 + 광역 조직 중심
```

### ⚠️ 4-b단계: EXT-8 비판적 자기검증 (EXT-1~4 후 반드시 실행)

```
실행 시점: EXT-1~4 초안 완성 직후, 최종 조립 전
적용: Q-N9

목적: {후보자명} 경력과 캠프 내부의 자기확증편향·자기중심적 사고를 식별하고 대안 제시

검증 프로세스:
1. EXT-1~4 초안을 "{후보자명}을/를 전혀 모르는 일반 유권자" 관점에서 다시 읽기
2. 5대 영역별 편향 진단:
   ① 브랜딩: "직업 프레임 없이도 매력적인가?"
   ② 이미지: "특정 메타포가 차갑지 않은가?"
   ③ 카피/슬로건: "캠프 전문용어가 유권자에게 전달되는가?"
   ④ 공약: "{핵심_공약1}을 모르는 유권자에게도 와닿는가?"
   ⑤ 경력 활용: "이력서가 아닌 해결한 문제로 설명 가능한가?"
3. 영역별 대안 A/B 제시
4. 최종 판정: 유지 / 부분수정 / 전면재검토

산출물:
- 편향 진단표: 5개 영역 × (현재/편향리스크/대안A/대안B)
- "유권자 눈 테스트" 결과
- 캠프 밖 시선 체크리스트 (내부자 전문용어·논리·전제 목록)
- 최종 권고 → EXT-1~4 초안에 대안 방향 병기로 반영
- 핵심 요약 2p에 편향 검증 결과 포함
```

## 5단계: Part 3 — 본선 참고사항

```
입력: INSIGHTS 본선 경쟁자 데이터 + 정치 상황
관점: "본선은 참고, 본선 경쟁력 프레이밍은 {선거모드} 무기"

생성:
REF-1. 본선 경쟁 구도 개요 (INSIGHTS 데이터 요약)
REF-2. "본선 경쟁력" 프레이밍 → {선거모드} 무기 활용법 (Q-N7)
REF-3. 정치 상황 변수 (정부 지지율/의석 구도/지역 진보-보수 구도)
REF-4. 경선→본선 전환 체크포인트 (72시간 내 전환)
```

## 6단계: INSIGHTS 교차 활용

| INSIGHTS 데이터 | 활용 위치 | {선거모드} 관점 |
|--------------|---------|-------------|
| 인지도/지지도 | PLAN 이중전략 | 목표 설정 + 본선 경쟁력 근거 |
| 감성 분석 | EXT-3 카피 | 긍정 키워드 강화 |
| 이슈 TOP10 | Q-N3 정합표 | 유권자 관심 이슈와 공약 매칭 |
| 지역별 분석 | FIELD 의제맵 | 권역별 유권자 특성 |
| 경쟁자 데이터 | REF 본선 참고 | 본선 경쟁력 프레이밍 근거 |
| SWOT | ME SWOT 교차 | 여론 기반 강점/약점 검증 |
| 전략 권고 | 전체 근거 | INSIGHTS 권고를 전략에 통합 |
| 리스크 | Q7 시나리오 | 리스크 추가 |

## 산출물 구조: DOCX, 한국어 고정, **50p 이내**

```
[핵심 요약 3p] ← 맨 앞
  1p: 전략 요약 ({선거종류}/{선거모드} 구도/승리공식/양자대결/브랜딩·슬로건 최종 추천)
  2p: 비판적 자기검증 결과 (5개 영역 편향진단 요약 + 핵심발견 3개)
  3p: 즉시 실행 액션 TOP10 (이번주/2주내/1개월내)

[Part 1: POLICY 보완] 15-17p
  ME / FIELD(1위 양자대결 포함) / PLAN / DO

[Part 2: 확장 섹션] 20-22p
  EXT-1~5: 브랜딩/이미지/카피/공약/콘텐츠
  EXT-6~7: 토론/조직 (선거모드 분기)
  EXT-8: 비판적 자기검증 (편향진단표 + 대안 + 유권자눈 테스트)

[Part 3: 본선 참고] 3-4p
  REF-1~4

[부록] 1-2p
  근거 인덱스 / 버전 로그
```
