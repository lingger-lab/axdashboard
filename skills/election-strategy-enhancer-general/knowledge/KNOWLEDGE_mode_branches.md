## primary_multi — 당내경선 (다자)

### 적용 조건
- 후보 3명 이상이 경선 참여
- 당원투표 + 여론조사 혼합 방식

### STEP 1 특화 섹션
```
(a) 승리 공식 — 당원투표(N%) + 여론조사(N%) 합산 공식
(b) 3개 시나리오 — 낙관/기본/비관
(c) 주차별 역전 경로 — 벤치마킹 공식 적용
(d) 전략공천 차단 조건 ★ primary_multi 전용
    - 차단 조건 1: 상승 모멘텀 증명 (주차별 지지율 상승 그래프)
    - 차단 조건 2: 본선 경쟁력 증명 (양자대결 시뮬레이션)
    - 차단 조건 3: 조직력 가시화 (DB/자원봉사 수치)
(e) 경쟁자 사퇴/단일화 시 표 이동 시뮬레이션 ★ primary_multi 전용
    - 각 경쟁자 사퇴 시 후보별 유입 비율 추정
    - 최악/기본/최선 시나리오 분리
```

### STEP 6 특화
```
(d) 전략공천 차단 패키지
    - 시나리오 1: 경선 직전 전략공천 시도 → 즉각 대응 프로토콜
    - 시나리오 2: 특정 후보 밀어주기 의혹 → 공개 성명 + 지지층 결집
    - 시나리오 3: 후보 단일화 압력 → 협상 조건 + 거부 시 대응
```

### 카피 방향
- 분열·분산 이미지 경계 → "우리의 힘을 모아" 통합 메시지
- 본선 경쟁력이 경선 승리 무기 ("경선 이기려면, 본선 이길 후보를")
- 전략공천 차단 메시지 ("국민이 후보를 선택해야 합니다")

## primary_bilateral — 당내양자경선

### 적용 조건
- 후보 2명만이 경선 참여
- 또는 다자 경선에서 양자 구도로 좁혀진 경우

### STEP 1 특화 섹션
```
(a) 0섬 게임 공식 ★ primary_bilateral 전용
    - 내 +1% = 상대 -1% (합산 100% 고정)
    - 따라서 "내 지지율 끌어올리기"와 "상대 지지율 끌어내리기"가 동치
    - 우선순위: 미결정층 공략 > 상대 지지층 이탈 유도
(b) 양자대결 시뮬레이션 (S1/S2/S3)
(c) 주차별 역전 경로
    주의: 전략공천 차단 섹션 없음
    주의: 단일화 섹션 없음
```

### STEP 5 특화
```
이슈 프레임: 상대 후보와 직접 비교
- 모든 이슈를 "나 vs 상대" 구조로 제시
- "상대는 [문제], 나는 [해법]" 반복 구조
```

### STEP 6 특화
```
(b) 1:1 집중 차별화
    - 상대와의 단일 비교축 설정
    - 토론 전략: 상대 약점을 직접 공략하는 질문 설계
    - 상대 공격 방어: 예상 공격 TOP5 + 즉답 스크립트
```

### STEP 7 특화
```
- 상대 진영 동향 모니터링 트랙 추가
- "상대 흔들기" 주간 목표 포함
```

### 카피 방향
- 2인 대결 → "선명한 대비" 메시지
- "A냐 B냐" 프레임 선점 ("이번 경선은 X vs Y의 선택입니다")
- 상대의 약점을 역으로 내 강점으로 ("그가 약한 부분이 내가 강한 이유")

## general_bilateral — 본선양자경선

### 적용 조건
- 공직선거법상 본선 선거
- 여당 vs 야당 1:1 구도

### STEP 1 특화 섹션
```
(a) 정당 지지율 기반 승리 공식 ★ general_bilateral 전용
    - 정당 지지율 (여당기반%) + 부동층(스윙보터%) + 후보 가산점(%)
    - 공식: 기본 정당 지지율 + 중도층 유입 목표 = 목표 득표율
(b) 스윙보터/중도층 공략 시뮬레이션
    - 중도층 비율 추정 (지역 특성 반영)
    - 중도층 N%를 가져오면 역전 가능한 임계점 계산
(c) 정치 지형 분석 (여야 지지율, 현 정부 지지율, 지역 정치 성향)
    주의: 전략공천 차단 없음, 당원투표 없음
```

### STEP 5 특화
```
이슈 프레임: 정당 프레임 vs 지역 프레임 선택
- 여당 후보: 정부 성과 연계 + 지역 현안 구체화
- 야당 후보: 정부 비판 + 대안 제시
- 공통: "지역민을 위한 일꾼" 프레임 (정당 색 희석)
```

### STEP 6 특화
```
(a) 본선 경쟁력 프레이밍
    - 여야 구도에서 이길 수 있는 논거
    - 정당 지지층 결집 + 중도층 흡수 전략
(b) 상대 정당 대비 차별화
    - 정책 차별화 (여당 vs 야당 공약 비교)
    - 인물 차별화 (경력, 자질, 비전)
(c) 반대 측 공세 예상 TOP5
    - 정당 이슈 공격 → 지역 이슈로 전환 방어 기술
```

### 준법 주의사항 (general_bilateral 전용)
```
⚠️ 본선 선거는 당내경선보다 선거법 적용이 엄격합니다.
- 사전선거운동 금지 기간 준수 필수
- 여론조사 결과 공표 제한 규정 확인
- 반드시 선거법 전문 변호사 자문 필수
```

## 모드 공통 — STEP 7 주차 계산 로직

```
campaign_days ÷ 7 → 올림 = 총 주차 수

D-19 (19일) → 3주
D-30 (30일) → 5주  ← 이전 버전 고정값, 이제 동적
D-45 (45일) → 7주
D-60 (60일) → 9주

주차별 테마 (최대 5주 기준, 이후는 반복 심화):
1주차: 인지도 구축 / 기반 다지기
2주차: 이슈 공략 / 차별화 강화
3주차: 역전 모멘텀 / 중도층 흡수
4주차: 1위 도전 / 핵심 지지층 결집
5주차: 막판 총력 / 투표 독려
```

## 모드 공통 — 벤치마킹 프레임 선택

|----------------|-------------|------------|
| underdog | 맘다니 2025 뉴욕시장 / 이준석 당대표 | "언더독 역전 공식" |
