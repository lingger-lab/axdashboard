# AI 선거전략 보고서 보완·확장 스킬 (일반화 v4.0)

## 핵심 원칙

1. **{선거모드} = 메인**, 반대 모드 = 참고사항
2. **비판이 아닌 보완**: 기존 POLICY 내용을 인정하면서 빠진 부분을 채움
3. **즉시 사용 가능한 산출물**: 카피, 수치, 일정, 매뉴얼 등 실전 콘텐츠 생성
4. **Q&A = 보완 프롬프트**: 점검 체크리스트가 아니라 "이 관점으로 새로 작성하라"
5. **파라미터 우선**: 입력된 파라미터가 문서 내용보다 항상 우선 바인딩됨

---

## STEP 0 — 작업 시작 시 반드시 실행: 파라미터 수집

보고서 생성 전, 아래 5개 항목을 순서대로 수집한다.
문서가 업로드된 경우에도 파라미터를 먼저 수집한 뒤 문서 분석을 시작한다.

### Step 1 — 선거 기본 정보
```
[선거종류] 시장 / 도지사 / 군수 / 구청장 / 국회의원 / 광역의원 / 기초의원 / 교육감 / 대통령 / 기타
[지역명]   OO시·도·구·군·선거구명
[선거모드] 당내경선 / 본선 / 경선+본선 통합
[경선방식] (경선 모드 해당 시) 당원투표 / 여론조사 / 혼합(당원:시민 = N:N) / 완전국민경선
[선거일]   YYYY-MM-DD (D-day 기준일)
```

### Step 2 — 후보자 정보
```
[후보자명]   성명
[소속 정당]  국민의힘 / 더불어민주당 / 기타
[주요 경력]  핵심 3줄 (없으면 "미입력" — 문서에서 자동 추출 시도)
[강점 키워드] 최대 5개 (없으면 문서 추출)
[슬로건]    (있으면 입력, 없으면 문서 추출 또는 AI 제안)
[핵심 리스크] 예상 공격 포인트 (없으면 "미입력")
```

### Step 3 — 경쟁자 정보
```
[경쟁자 수]   1~10명
[경쟁자 목록] 이름 + 인지도 순위(선택) + 핵심 특성(선택)
※ 미입력 시 "경쟁자 {N명} 미상" 상태로 진행, 문서에서 자동 추출 시도
```

### Step 4 — 지역 권역 정보
```
[권역1] / [권역2] / [권역3] — 지역 내 주요 행정 권역
※ 미입력 시 "주요 권역" 통칭 사용
```

### Step 5 — 보고서 업로드 (선택)
```
INSIGHTS 여론분석 보고서 (PDF/DOCX) — 선택
POLICY 전략분석 보고서 (PDF/DOCX)   — 선택
기타 참고 자료 (여론조사 결과, 공약집 등) — 선택
```

### 입력 조합별 처리 방식
| 입력 조합 | 처리 방식 | 모드명 |
|----------|---------|------|
| 파라미터 + 문서 | 문서 내용 우선 + 파라미터로 보완 | 완전 모드 (권장) |
| 파라미터만 | 기본 템플릿 + 입력값으로 생성 | 신속 모드 |
| 문서만 | 문서에서 자동 추출 시도 | 추출 모드 |
| 모두 없음 | 5개 항목 수집 후 시작 | 인터랙티브 모드 |

## 파라미터 바인딩 규칙

수집된 파라미터는 아래 플레이스홀더로 문서 전체에 자동 치환된다.

| 플레이스홀더 | 바인딩 값 |
|------------|---------|
| `{후보자명}` | Step 2 입력값 |
| `{지역명}` | Step 1 입력값 |
| `{선거종류}` | Step 1 입력값 |
| `{선거모드}` | Step 1 입력값 |
| `{경선방식}` | Step 1 입력값 |
| `{경쟁자1}` ~ `{경쟁자N}` | Step 3 입력값 |
| `{권역1}` / `{권역2}` / `{권역3}` | Step 4 입력값 |
| `{세그먼트1}` ~ `{세그먼트4}` | 문서 추출 or AI 자동 설정 |
| `{핵심_공약1}` ~ `{N}` | 문서 추출 or Step 2 입력값 |
| `{핵심_이슈_수치}` | 문서 추출 or 사용자 입력 |
| `{후보자_슬로건}` | Step 2 입력값 or 문서 추출 |
| `{후보자_리스크}` | Step 2 입력값 or 문서 추출 |

## 선거종류별 전략 분기

| 선거종류 | 경선방식 주요 옵션 | 주요 경쟁 구도 | 전략 특성 |
|--------|----------------|-------------|---------|
| 광역단체장(시장·도지사) | 당원+여론조사 혼합 | 다자경선 → 양자구도 | 조직력 + 본선 경쟁력 병행 |
| 기초단체장(군수·구청장) | 당원 중심 or 완전국민 | 3~5인 경선 | 지역밀착 의제 우선 |
| 국회의원 | 당원+국민 혼합 | 1~3인 경선 | 지역 현안 + 당론 연계 |
| 광역·기초의원 | 당원 중심 | 2~4인 경선 | 생활정치 의제 집중 |
| 교육감 | 단일 선출(경선 없음) | 본선 다자구도 | 정치색 탈색 필요 |
| 대통령 | 완전국민경선 | 전국 광역 조직 | 이념+인물+정책 종합 |

## 입력물 식별

### INSIGHTS 보고서 (여론분석)
식별: "여론분석 보고서", "CampOne", "감성 분석"
구조 (11개 섹션): 종합요약 / 데이터수집 / 감성분석 / 인지도·지지도 / 추세 / 이슈 / 타임라인 / 지역 / SWOT / 리스크 / 전략권고

### POLICY 보고서 (전략분석)
식별: "선거 전략 분석 보고서", "SWOT 분석", "승리 공식"
구조 (4개 대섹션 + 부록):
1. **ME** — 프로필, 강점/약점, 차별화, 보완필요, SWOT, 포지셔닝
2. **FIELD** — 선거구 분석, 경쟁자 상세, PEST
3. **PLAN** — 승리공식, 표밭 세그먼트, 메시지전략, 로드맵
4. **DO** — 세그먼트 공략, 정책수립, 콘텐츠방향
5. **부록** — 브랜딩요약, 여론분석요약, 핵심정책목록

## 확장 섹션 (Part 2)

### [EXT-1] 개인 브랜딩 전략
Q1, Q9, Q11 적용 | "유권자가 봤을 때 매력적인" 브랜딩
→ 정체성(What/How/Why) / 시그니처 스토리 3종 / 키워드+금지어 / 신뢰자산 / 경쟁자 대비 차별점

### [EXT-2] 이미지메이킹 전략
Q1, Q11 적용 | 유권자 접점에서의 이미지
→ 비주얼·언어 아이덴티티 / 상황별 이미지 코드 / 복장·촬영 가이드

### [EXT-3] 브랜딩 카피 체계
Q11, Q1, Q9 적용
→ 슬로건 / 서브카피 / 원라이너 / 유권자FAQ / 경쟁자공격 대응 / 소셜카피 / 여론조사용 시민 메시지

### [EXT-4] 공약 구체화
Q4, Q3, Q8 적용 | "유권자가 납득하는" 실행가능성
→ 가능/검토/불가 / 실행카드 / Before→After / 경쟁자 공약 대비 / 검증표

### [EXT-5] 콘텐츠 채널·기획
Q12, Q13, Q11 적용
→ 채널 매트릭스 / 유형5종 / 리퍼포징 / 캘린더 / CIVIC HUB / 현장-디지털싱크 / AI가이드

### [EXT-6] 토론/합동연설회 전략 🆕
Q-N5, Q9, Q-N2 적용
→ 예상주제TOP5 / 답변템플릿(팩트→원칙→조치→시간표) / 경쟁자별 대비 / 3분 구조 / 위기답변

> **선거모드 분기**
> - 경선 모드: 합동연설회·경선 토론 중심
> - 본선 모드: 공중파 토론·정책 토론 중심으로 내용 교체

### [EXT-7] 조직 운영 체계 🆕
Q-N6, Q5, Q-N1 적용
→ DB관리 / 간담회매뉴얼 / 전화문자스크립트 / 자원봉사관리 / 주3회실행표 / 불만TOP5+답변

> **선거모드 분기**
> - 경선 모드: 당원 DB + 당원 접점 중심
> - 본선 모드: 일반 시민 자원봉사자 + 광역 조직 중심

### [EXT-8] 비판적 자기검증 — "이게 최선인가?" 🆕
**Q-N9** 적용 | **EXT-1~4 초안 완성 후, 최종 조립 전 반드시 실행**

| 검증 대상 | 확증편향 리스크 | 검증 질문 |
|---------|-------------|---------|
| 개인 브랜딩 | 후보자 직업/경력에 과도 의존 | 직업 프레임 없이도 브랜딩이 성립하는가? |
| 이미지메이킹 | 특정 메타포 고착 → 유권자에게 다르게 느껴질 수 있음 | 메타포 없이 이미지가 전달되는가? |
| 브랜딩 카피 | 캠프 내부 언어 → 시민 체감과 괴리 | 시민이 일상에서 쓰는 말로 바꾸면? |
| 슬로건 | 위기 강조가 희망보다 강할 수 있음 | 위기 vs 희망 프레임, 어느 쪽이 유리한가? |
| 주요 공약 | 캠프의 자부심 → 유권자 인지도 낮을 수 있음 | 모르는 유권자에게 전달되는가? 번역이 필요한가? |

산출물:
- **편향 진단표**: 5개 영역 × (현재 방향 / 편향 리스크 / 대안 A / 대안 B)
- **유권자 눈 테스트**: "{후보자명}을/를 전혀 모르는 일반 유권자" 관점 검증
- **캠프 밖 시선 체크리스트**: 내부자만 이해하는 용어/논리/전제 목록
- **최종 권고**: 각 영역별 "유지 / 부분수정 / 전면재검토" 판정
