# AI 선거전략 보고서 보완·확장 스킬 (경선 중심 v3.2)

## 개요

CampOne AI의 INSIGHTS/POLICY 보고서를 업로드받아, Q&A 지식베이스(`references/qa_knowledge_base.md`, 27개 항목)의 **질문(Q)**으로 LLM이 업로드된 문서를 비판적으로 참고하여 **새로운 보완 답변(A)**을 생성한다.

### 핵심 원칙

1. **경선(Primary) = 메인**, 본선(General) = 참고사항
2. **비판이 아닌 보완**: 기존 POLICY 내용을 인정하면서 빠진 부분을 채움
3. **즉시 사용 가능한 산출물**: 카피, 수치, 일정, 매뉴얼 등 실전 콘텐츠 생성
4. **Q&A = 보완 프롬프트**: 점검 체크리스트가 아니라 "이 관점으로 새로 작성하라"

### 경선/본선 프레임

```
[메인] 경선 승리 전략 (50/50)
├── 경선 경쟁자 5인 분석이 중심
├── 당원 세그먼트별 공략이 핵심
├── 당원50%(조직) + 시민50%(여론) 이중전략
├── 당원 대상 브랜딩·카피·콘텐츠가 우선
│
└── [참고] 본선 고려사항
    ├── 본선 경쟁자 개요 (INSIGHTS 데이터)
    ├── "본선 경쟁력" 프레이밍 → 경선 무기로 활용
    ├── 정치 상황 변수 (여대야소, 정부 지지율 등)
    └── 경선→본선 전환 체크포인트
```

## 입력물 식별

### INSIGHTS 보고서 (여론분석)
식별: "여론분석 보고서", "CampOne", "감성 분석"
구조 (11개 섹션): 종합요약 / 데이터수집 / 감성분석 / 인지도·지지도 / 추세 / 이슈 / 타임라인 / 지역 / SWOT / 리스크 / 전략권고

### POLICY 보고서 (전략분석)
식별: "선거 전략 분석 보고서", "SWOT 분석", "승리 공식"
구조 (4개 대섹션 + 부록):
1. **ME** — 프로필, 초기강점/약점, 차별화, 보완필요, SWOT(SO/WO/ST/WT), 포지셔닝(문구/타깃/경쟁자비교)
2. **FIELD** — 선거구(인구통계/지역특성/정치성향/지역현안), 경쟁자5인상세(인지도/강점/약점/전략/대응), PEST
3. **PLAN** — 승리공식(득표목표/표밭4세그먼트/핵심전략/전제조건/리스크대응), 메시지전략(슬로건/세그먼트별메시지/해시태그), 로드맵4단계(준비기/확산기/확대기/마무리기+임계경로)
4. **DO** — 세그먼트공략(연령/지역/계층별), 세그먼트별정책수립(문제/해결/기대효과), 콘텐츠방향(전체전략/세그먼트별/채널별/바이럴)
5. **부록** — 브랜딩요약(슬로건3종/10대공약), 여론분석요약, 핵심정책목록

### [EXT-8] 비판적 자기검증 — "이게 최선인가?" 🆕
**Q-N9** 적용 | **보고서 최종 작성 직전에 반드시 실행**

후보자 경력과 캠프 내부에서 오는 **자기확증편향(Confirmation Bias)**과 **자기중심적 사고(Egocentric Bias)**를 식별하고, 현재 전략이 정말 최선인지 검증한다. 각 항목에 대해 "현재 방향"과 "대안 방향"을 병렬 제시한다.

| 검증 대상 | 확증편향 리스크 | 검증 질문 |
|----------|--------------|----------|
| 개인 브랜딩 | "의사 출신"에 과도 의존 → 유권자는 의사보다 시장을 뽑는다 | 의료 프레임 없이도 브랜딩이 성립하는가? 대안 브랜딩 축은? |
| 이미지메이킹 | "진단·처방" 메타포 고착 → 유권자에게 차갑게 느껴질 수 있음 | "치료" 대신 "함께 만드는" 이미지가 더 효과적이지 않은가? |
| 브랜딩 카피 | "골든타임" "복합 쇠퇴 증후군" = 캠프 내부 언어 → 시민 체감과 괴리 | 시민이 일상에서 쓰는 말로 바꾸면 어떻게 되는가? |
| 슬로건 | "100만 창원, 다시 일으킵니다" → 위기 강조가 희망보다 강함 | 위기 프레임 vs 희망 프레임, 어느 쪽이 당원 전환에 유리한가? |
| 주요 공약 | "피지컬AI 허브" = 캠프의 자부심 → 당원/시민 인지도 낮을 수 있음 | 피지컬AI를 모르는 유권자에게 이 공약이 전달되는가? 번역이 필요한가? |

산출물:
- **편향 진단표**: 5개 핵심 영역 × (현재 방향 / 편향 리스크 / 대안 방향)
- **"만약 다르게 했다면" 시나리오**: 각 영역의 대안 방향을 실제로 적용한 버전
- **유권자 눈 테스트**: "후보를 전혀 모르는 50대 당원이 처음 봤을 때" 관점 검증
- **캠프 밖 시선 체크리스트**: 내부자만 이해하는 용어/논리/전제가 있는가?
