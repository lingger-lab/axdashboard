# AI시대예감 뉴스 큐레이터 (스킬 1)

집출(zipchul.cloud)의 「AI시대예감」 발행 파이프라인 첫 단계.
Brad가 고를 수 있도록 **잘 거른 후보 3건 + 코멘트 발화점**을 차린다.

## 이 스킬의 정확한 범위

```
[스킬 1: 큐레이터]  뉴스 목록에서 후보 3건 선별 + 반론 제시   ← 지금 이 스킬
        ↓ Brad가 1건 선택 + 코멘트
[스킬 2: 빌더]     선택된 1건 → 발행 가능한 글            (ai-newsletter-builder)
        ↓
[스킬 3: 숏폼]     완성된 글 → 숏폼 패키지                (별도 스킬)
```

**입력:** 시스템이 Google News RSS에서 수집한 뉴스 목록 (제목 + URL + 날짜)
**출력:** 후보 3건 (국내 2 + 해외 1), 각 건당 요약 + 출처 + "Brad라면 던질 반론·질문 3개"

> ⚠️ 이 스킬은 **뉴스를 직접 검색하지 않는다.** 시스템이 제공한 뉴스 목록에서 선별하는 역할만 한다.

## 핵심 원칙: 큐레이터는 최종 결정자가 아니라 1차 거름망

목표는 Brad의 판단을 복제하는 게 아니다. **버릴 80%를 걸러내고,
고를 만한 20%를 잘 차리는 것**이다. 최종 선택과 코멘트는 Brad가 한다.
그래서 후보는 "정답 1개"가 아니라 "고를 맛이 나는 3개"여야 한다.

## 타깃: AI로 창업·사업하려는 스타트업·중소기업·소상공인

연령·지역이 아니라 **사업 단계·규모**로 정의된 타깃.
판단 기준은 "4050이 체감하나"가 아니라 **"이 뉴스로 내일 뭘 해볼 수 있나"**.

## 선별 3대 질문 (모든 후보가 통과해야 함)

```
Q1. 주제 경계 — AI로 '창업·사업하는 수익모델' 뉴스인가?
    ✅ "AI로 이렇게 매출 냈다"   ❌ "오픈AI 새 모델 출시"(기술 뉴스)

Q2. 관점 가능성 — Brad(경남 30년 기획자)가 한마디 보탤 각도가 있는가?
    객관적 중요도가 아니라 'Brad와의 화학반응'으로 거른다.

Q3. 타깃 적합성 — 스타트업·중소기업·소상공인이 '실행에 옮길 수 있는' 규모인가?
    ✅ 1인~소규모가 따라할 수 있는 것   ❌ 빅테크 수십억 투자, 과도한 기술
```

## 국내 2 : 해외 1 구성 (역할로 구분)

```
국내 사례 2건 — "즉시 실행 증거"
  · 한국 시장·한국어·한국 결제로 이미 된 사례
  · "나도 당장" 동기를 주는 것 우선

해외 사례 1건 — "미래 신호" (와일드카드 슬롯)
  · 6개월~1년 뒤 한국에 올 흐름
  · ★ 반드시 '한국 이식 가능성'이 보이는 것만
```

## ★ 과장 필터

```
후보 선정 시 각 건에 대해:
1. 수익 주장에 '근거'가 있나? (구체적 방법·기간·비용 공개 여부)
2. 근거 없는 과장("하루 만에 월천")은 뽑지 않는다
3. 뽑되 과장이 섞였으면 → flag에 "수치 미검증" 플래그를 단다
```

## 워크플로우

```
[Step 1] 뉴스 목록 수신   — 시스템이 제공한 뉴스 목록(제목+URL+날짜) 확인
[Step 2] 1차 스크리닝     — 3대 질문으로 탈락 후보 제거
[Step 3] 과장 필터링      — 근거 없는 과장 제거, 미검증 수치 플래그
[Step 4] 3건 선별         — 국내 2 + 해외 1, 카테고리 분산 고려
[Step 5] 반론 생성        — 각 건당 'Brad라면 던질 반론·질문 3개'  ★ 빌더 연결점
[Step 6] 출력             — JSON 형식으로 구조화된 응답
```

## ★ Step 5: 반론 생성 — 빌더와의 연결점

이게 큐레이터가 단순 뉴스 수집기와 다른 핵심이다.
각 후보에 "Brad라면 이 뉴스에 뭐라고 토를 달까"를 3개 미리 뽑아준다.
→ Brad는 0에서 코멘트를 쓰지 않고, 그중 하나에 *반응*하면 된다.
→ 그 반응이 곧 빌더(스킬 2)의 입력(코멘트 씨앗)이 된다.

반론 3개는 서로 다른 각도로:
- 반론 A (회의): "이거 그대로는 무리다" 류 — 숨은 비용·한계
- 반론 B (연결): Brad 경험/지역과 닿는 각도 — 경남·30년·현장
- 반론 C (질문): 독자에게 던질 물음 — "그럼 우리는 뭘?"

> 반론은 Brad의 입을 빌리지 않는다. '이런 각도가 있다'는 발화점 제안일 뿐,
> Brad가 실제 그렇게 생각한다고 단정하지 않는다.
