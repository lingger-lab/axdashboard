# 한국 선거 맥락 참조 자료

분석 시 절대 제약과 도전 가능한 관행을 구별하는 데 사용합니다.

---

## 절대 제약 (변경 불가능)

### 법적 제약
- 선거일은 법정 기일 (지방선거: 4년마다 6월 첫째 수요일)
- 선거운동 기간: 공식 선거운동 기간 제한 (통상 13일)
- 선거비용 제한: 선거구별 법정 상한액
- 기부행위 제한: 선거구민에 대한 기부행위 금지
- 사전투표: 선거일 전 금~토 2일간 실시
- 여론조사 공표 금지: 선거일 전 6일간
- 당내경선: 각 정당 규정에 따라 방식 결정 (ARS, 투표 등)

### 구조적 제약
- 선거구 획정: 법정 기준에 따라 확정
- 유권자 수: 인구통계학적으로 결정
- 정당 공천: 각 정당의 공천 규정에 따름
- 투표소 위치: 선거관리위원회가 결정
- ARS 경선: 무작위 추출 → 전체 모집단 지지율이 곧 득표율 (통계적 사실)

## 도전 가능한 관행

### 선거운동 관행

| 관행 | 일반적 이유 | 도전 근거 |
|------|-----------|----------|
| 거리 유세 중심 | "유권자를 직접 만나야" | 도달률 대비 효율 미검증 |
| 현수막/명함 대량 배포 | "인지도를 높여야" | 인지도→지지율 전환율 미측정 |
| 조찬/만찬 모임 | "핵심 지지층 관리" | 실제 표 전환 효과 불명 |
| SNS 일방향 홍보 | "젊은 층 공략" | 도달 vs 공감 vs 행동 구분 부재 |
| 전화 여론조사 의존 | "과학적 분석" | 응답률 하락, 표본 편향 증가 |
| DB 전수 접촉 | "많이 접촉할수록 유리" | 확정 지지자 반복 접촉은 낭비 |
| 외국 사례 벤치마킹 | "검증된 전략" | 선거 구조 차이 미반영 |

### 여론분석 관행

| 관행 | 문제점 |
|------|--------|
| 지지율만 추적 | 투표 의향 강도, 이탈 가능성 미측정 |
| 연령/성별 교차분석 | 라이프스타일/가치관 기반 세분화 부재 |
| 경쟁자 비교만 수행 | 무관심층/미결정층 분석 부족 |
| 시계열 단순 비교 | 외부 이벤트 영향 통제 부재 |
| 단순 인지도 측정 | 인지도 ≠ 회상 우선순위(recall priority) |

## 유권자 행동 패턴 (제1원칙 분석 시 유의점)

### 흔한 미검증 가정 5가지

1. "높은 인지도 = 높은 지지율" → 인지도와 호감도는 별개
2. "정책이 좋으면 표가 온다" → 정책 인지율 자체가 매우 낮음
3. "야당 지지자는 설득 불가" → 이슈별 교차 지지 존재
4. "투표율이 높으면 유리/불리" → 선거구별 역학이 다름
5. "온라인 여론 = 실제 여론" → 온라인 활동 인구 편향

### 실제 투표 의사결정 요인 (연구 기반)

- **후보자 신뢰도와 진정성**: 정책보다 우선하는 경우 많음
- **소속 정당**: 한국 선거에서 가장 강력한 예측 변수
- **주변인 영향**: 가족, 친구, 직장 동료
- **현직 효과**: 현직자 인지도 프리미엄
- **투표일 편의성**: 사전투표 도입 이후 변화

### 당내경선 특수 역학

- 당원 충성도가 일반 유권자보다 높음 → 파벌/인맥이 정책보다 강력한 변수
- ARS 경선은 무작위 → "동원"이 아니라 "전체 지지율"이 핵심
- 경선 투표자는 정치 관여도가 높음 → 이슈 민감도가 높지만 정당 정체성도 강함
- 다자 경선(5인+)은 표 분산 → "유일한 OO" 포지셔닝이 유리

## 2026 지방선거 특수 맥락

분석 시점의 정치 환경에 따라 **웹 검색으로 최신 정보를 보강**해야 합니다.

**확인 필요 항목:**
- 현 정부 지지율 추세 (여야 역학에 영향)
- 주요 정치 이슈 (경제, 안보, 사회 이슈)
- 지역별 정당 지지 구도 변화
- 공천 일정 및 주요 후보 동향
- 선거법 개정 사항
- 당내경선 방식 확정 여부 (ARS, 투표, 혼합 등)

## 한국 경선 역전 사례 (벤치마킹 참고)

외국 사례보다 동일 선거 구조의 한국 사례가 더 직접적인 교훈을 제공합니다:

| 사례 | 핵심 역전 요인 | 교훈 |
|------|-------------|------|
| 이준석 2021 당대표 경선 | 온라인 언더독 + 세대 프레이밍 | 디지털 채널이 당원에게도 영향 |
| 안철수 2022 대선 경선 | 단일화 레버리지 | 전략적 포지셔닝이 지지율보다 중요 |
| 원희룡 2022 대선 경선 | 거리 유세 집중 + 토론 퍼포먼스 | 결정적 이벤트 1~2개가 흐름 전환 |

**공통 패턴**: ARS/투표 경선에서의 역전은 "결정적 1~2개 이벤트"에 의해 발생. 선형적 상승이 아닌 계단식(특정 이벤트 후 급등) 패턴이 일반적.
